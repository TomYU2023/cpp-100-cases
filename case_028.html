find_if 条件筛选

新语法讲解

remove_if 和 erase-remove 惯用法

remove_if 并不是真的删除元素，它只是把不满足条件的元素移到前面，返回新的逻辑终点。必须配合 erase 才能真正删除。这是 C++ 社区著名的 erase-remove idiom。C++20 引入了 std::erase_if 一步搞定。

举一反三

// C++17 及之前
v.erase(remove_if(v.begin(),v.end(), pred), v.end());
// C++20 一步
erase_if(v, pred); // 对所有标准容器适用

any_of / all_of / none_of

三个谓词判断算法，都是短路求值：any_of 遇到第一个 true 就停止，all_of 遇到第一个 false 停止，none_of 遇到第一个 true 停止。不需要 O(n) 就能提前结束。

举一反三

vector v = {2,4,6,7,8};
all_of(v.begin(),v.end(), [](int n){return n%2==0;});
// 遇到 7 就返回 false，不看后面的 8

举一反三

用 find_if 实现二分查找的通用版

auto it = find_if(v.begin(),v.end(),bind(greater<>(),_1,10));
// 找第一个 >10 的元素（无序数组）

bind 绑定谓词参数

C++20 ranges::find_if

#include 
auto it = ranges::find_if(v, [](int n){return n>10;});
// 不传 begin/end，直接对容器操作

range 版本更简洁

partition 稳定分区

auto mid = stable_partition(v.begin(), v.end(),
  [](int n){return n%2==0;});
// 偶数在前，奇数在后，各自内部保持原序

stable_partition 保证同组内相对顺序不变
