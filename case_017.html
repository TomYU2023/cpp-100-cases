新语法讲解

Lambda 表达式完整语法

[捕获列表](参数列表) mutable -> 返回类型 { 函数体 }。常见捕获方式：[] 不捕获、[=] 按值全捕获、[&] 按引用全捕获、[this] 捕获当前对象指针。
[x]   // 只按值捕获 x
[&x]  // 只按引用捕获 x
[=, &y] // 除 y 引用外其余按值
[this] // 类成员函数中使用，捕获 this

mutable 关键字

默认情况下，按值捕获的变量在 lambda 内不可修改（const）。加 mutable 后可以修改，但只影响 lambda 内部的副本，不影响外部变量。
int n = 0;
auto f = [n]() mutable { ++n; return n; };
f(); // 返回 1
f(); // 返回 2
cout << n; // 仍是 0

举一反三

用 lambda 实现回调

代码：void doTask(function cb) { cb(42); }

思路：把 lambda 当回调函数传递

捕获 this 在类中使用

代码：class Timer { void start() { auto cb=[this]{ onTick(); }; } };

思路：[this] 让 lambda 访问类成员

泛型 lambda (C++14)

代码：auto add=[](auto a, auto b){ return a+b; };

思路：auto 参数类型由编译器推导
