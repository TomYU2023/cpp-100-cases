案例 2：猜数字游戏

生成一个随机数让用户猜测，每次反馈偏大或偏小，直到猜中为止。学习随机数和条件循环。

核心知识点：rand/srand、while 循环、条件判断

完整代码（含逐行注释）

// 案例 2：猜数字游戏
// 生成 1~100 随机数，用户猜测直至猜中
#include <iostream>
#include <cstdlib>   // rand() 和 srand()
#include <ctime>     // time() 用于随机种子
using namespace std;

int main() {
    // 使用当前时间作为随机数种子，确保每次运行结果不同
    srand(static_cast<unsigned>(time(0)));
    // 生成 1~100 之间的随机数
    int target = rand() % 100 + 1;
    int guess;
    int attempts = 0; // 记录猜测次数

    cout << "我已想好了一个 1~100 之间的数字，请猜！" << endl;

    // 无限循环，直到猜中才 break
    while (true) {
        cout << "请输入你的猜测：";
        cin >> guess;
        attempts++; // 每猜一次计数 +1

        if (guess > target) {
            cout << "太大了，再试试！" << endl;
        } else if (guess < target) {
            cout << "太小了，再试试！" << endl;
        } else {
            cout << "恭喜你猜中了！共用了 " << attempts << " 次。" << endl;
            break; // 猜中后退出循环
        }
    }
    return 0;
}

语法规则：rand() 和 srand()

#include <cstdlib>
srand(time(0));
int r = rand() % 100 + 1; // 1~100

举一反三：rand() 质量较低，C++11 后推荐使用 中的 mt19937 和 uniform_int_distribution。

语法规则：while(true) 无限循环

while (true) {
    // 循环体
    if (condition) break; // 必须提供退出条件
}

举一反三：也可用 while(条件)、do...while 实现。break 退出当前循环，continue 跳过本次剩余语句进入下一轮。

语法规则：if-else if-else 链

if (a > b) { } else if (a < b) { } else { }

举一反三：条件互斥时用 else if 而非多个独立 if，效率和逻辑都更好。注意所有分支都应初始化使用的变量。
