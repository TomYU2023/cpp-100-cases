新语法讲解

cin 输入流

`cin >> 变量` 从标准输入（键盘）读取数据。`>>` 是流提取运算符。默认以空格/换行分隔。
int a; double b;
cin >> a >> b;  // 连续读取两个值

随机数生成

C++ 标准库的随机数：`rand()` 返回 0~RAND_MAX 的整数。`rand() % N` 得到 0~N-1。现代 C++ 推荐用 `` 库。
#include 
random_device rd;
mt19937 gen(rd());
uniform_int_distribution<> dis(1, 100);
int x = dis(gen);

类型转换 static_cast

`static_cast<目标类型>(值)` 是 C++ 的显式类型转换，在编译期执行，比 C 风格 `(类型)值` 更安全。
double d = 3.14;
int i = static_cast(d);  // i = 3
