新语法讲解

函数定义与调用

`返回类型 函数名(参数类型 参数名, ...) { 函数体; return 返回值; }`。函数是代码复用的基本单元。
int max(int a, int b) {
    return a > b ? a : b;
}
int m = max(10, 20);  // m = 20

switch 语句

`switch(表达式) { case 值: ... break; default: ... }` — 多分支选择结构，比多个 if-else 更清晰。
switch (grade) {
    case 'A': cout << "优秀"; break;
    case 'B': cout << "良好"; break;
    default: cout << "其他";
}

函数内联 inline

短小的函数建议加 `inline` 关键字，编译器会尝试把函数体直接嵌入调用处，减少调用开销。
inline int square(int x) { return x * x; }
