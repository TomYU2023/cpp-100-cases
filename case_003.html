案例 3：简易计算器

实现一个支持加减乘除的简易计算器，使用 switch 语句处理运算符选择。

核心知识点：switch-case、cin 输入、浮点数运算

完整代码（含逐行注释）

// 案例 3：简易计算器
// 支持加减乘除四种运算
#include <iostream>
using namespace std;

int main() {
    double num1, num2, result;
    char op;

    cout << "简易计算器" << endl;
    cout << "请输入表达式（如 3 + 5）：";
    cin >> num1 >> op >> num2;

    // switch 语句根据运算符执行对应操作
    switch (op) {
        case '+':
            result = num1 + num2;
            break;
        case '-':
            result = num1 - num2;
            break;
        case '*':
            result = num1 * num2;
            break;
        case '/':
            // 除法需要检查除数是否为零
            if (num2 != 0) {
                result = num1 / num2;
            } else {
                cout << "错误：除数不能为零！" << endl;
                return 1; // 异常退出
            }
            break;
        default:
            cout << "错误：不支持的运算符！" << endl;
            return 1;
    }

    cout << num1 << " " << op << " " << num2 << " = " << result << endl;
    return 0;
}

语法规则：switch-case 语句

switch (op) {
    case '+': /* ... */ break;
    case '-': /* ... */ break;
    default: /* ... */
}

举一反三：switch 只能判断整数或枚举类型。每个 case 必须以 break 结尾，否则会「穿透」执行下一个 case。

语法规则：除零检查

if (num2 != 0) { result = num1 / num2; } else { cout << "错误" << endl; }

举一反三：整数除法还会截断小数。浮点数除法用近乎零判断：fabs(num2) < 1e-9。

语法规则：cin 连续输入

cin >> num1 >> op >> num2;

举一反三：>> 运算符默认以空白字符（空格/换行/Tab）分隔。输入类型不匹配时 cin 会进入 fail 状态，可用 cin.clear() 恢复。
