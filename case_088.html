案例 88：std::lexicographical_compare 字典序

使用字典序比较两个序列，实现类似字符串的比较逻辑。

核心知识点：lexicographical_compare、字典序比较、自定义排序

完整代码（含逐行注释）

// 案例 88：std::lexicographical_compare 字典序
// 按字典序比较两个序列
#include <iostream>
#include <algorithm>
#include <vector>
#include <string>
using namespace std;

int main() {
    // 1. 数字序列的字典序比较
    vector<int> A = {1, 2, 3, 4};
    vector<int> B = {1, 2, 4, 0};
    vector<int> C = {1, 2, 3};

    cout << "A: ";
    for (int x : A) cout << x << " ";
    cout << "\nB: ";
    for (int x : B) cout << x << " ";
    cout << "\nC: ";
    for (int x : C) cout << x << " ";
    cout << endl;

    cout << "\nA < B: " << lexicographical_compare(
        A.begin(), A.end(), B.begin(), B.end()) << endl;
    cout << "C < A: " << lexicographical_compare(
        C.begin(), C.end(), A.begin(), A.end())
         << " (C 是 A 的前缀)" << endl;

    // 2. 自定义比较规则
    // 按绝对值字典序比较
    vector<int> X = {1, -5, 3};
    vector<int> Y = {1, 4, 2};

    bool lessByAbs = lexicographical_compare(
        X.begin(), X.end(), Y.begin(), Y.end(),
        [](int a, int b) { return abs(a) < abs(b); });
    cout << "\n按绝对值: X < Y = " << lessByAbs << endl;

    // 3. 版本号比较
    // 1.2.3 vs 1.2.10: 按分段数字比较
    vector<int> v1 = {1, 2, 3};
    vector<int> v2_10 = {1, 2, 10};
    vector<int> v2 = {1, 2, 3, 0};

    cout << "\n版本号比较:" << endl;
    cout << "1.2.3 < 1.2.10: " << lexicographical_compare(
        v1.begin(), v1.end(), v2_10.begin(), v2_10.end()) << endl;
    cout << "1.2.3 < 1.2.3.0: " << lexicographical_compare(
        v1.begin(), v1.end(), v2.begin(), v2.end())
         << " (前缀更短算更小)" << endl;

    // 4. 实际应用：多列排序的基础
    // 比较两个坐标：先比 x，再比 y
    using Point = pair<int, int>;
    auto cmp = [](const Point& p1, const Point& p2) {
        return lexicographical_compare(
            &p1.first, &p1.first + 2,
            &p2.first, &p2.first + 2);
    };

    Point p1{1, 5};
    Point p2{1, 3};
    cout << "\nPoint(1,5) < Point(1,3): " << cmp(p1, p2) << endl;

    return 0;
}

语法规则：std::lexicographical_compare

bool result = lexicographical_compare(
    A.begin(), A.end(), B.begin(), B.end());
// 逐元素比较，遇到第一对不等的元素时按该对比较
// 一个序列是另一个的前缀时，短的更小
// 类似字符串的 strcmp

举一反三：字典序是序列比较的标准方式。与字符串比较逻辑一致。空序列小于任何非空序列。默认用 operator< 比较元素。

语法规则：自定义比较器

lexicographical_compare(A.begin(), A.end(),
    B.begin(), B.end(), customLess);
// 按自定义规则逐元素比较
// 可比较复杂类型（如按结构体某个字段）

举一反三：自定义比较器可用于按不同字段排序比较。如按日期排序（先比年，再比月，再比日）。比较器必须满足严格弱序（strict weak ordering）。

语法规则：版本号比较

// 版本号 「1.2.10」 vs 「1.2.3」
// 不能用字符串比较（「10」 < 「3」）
// 拆分为数字序列后字典序比较
auto v1 = split("1.2.10"); // {1,2,10}
auto v2 = split("1.2.3");  // {1,2,3}

举一反三：版本号分段字典序是 lexicographical_compare 的经典应用。注意补零处理（1.2 = 1.2.0）。也可用 tuple 比较（std::tie）实现多列排序。
