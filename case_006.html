新语法讲解

new 与 delete

`new 类型[N]` 在堆上分配 N 个连续对象，返回首地址指针。必须用 `delete[]` 释放，否则内存泄漏。`new 类型` 分配单个对象，用 `delete` 释放。
int* p = new int(42);    // 单个 int
delete p;
int* arr = new int[100]; // 数组
delete[] arr;

构造函数与初始化列表

`类名(参数) : 成员1(值1), 成员2(值2) { 函数体 }`。初始化列表在函数体之前执行，对 const 成员和引用成员必须用初始化列表。
class Point {
    int x, y;
public:
    Point(int a, int b) : x(a), y(b) {}
};

const 成员函数

在函数声明末尾加 `const`，表示该函数不会修改对象状态。const 对象只能调用 const 成员函数。
int getValue() const { return value; }
