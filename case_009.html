新语法讲解

struct 结构体

`struct 名称 { 成员列表 };` — C++ 中 struct 和 class 几乎一样，唯一区别是 struct 默认 public、class 默认 private。struct 通常用于纯数据聚合。
struct Point { int x, y; };
Point p = {3, 5};
cout << p.x;  // 3

Lambda 表达式

`[捕获列表](参数列表) -> 返回类型 { 函数体 }`。C++11 引入的匿名函数对象，常作为回调或比较器。返回类型可省略（自动推导）。
auto add = [](int a, int b) { return a + b; };
cout << add(3, 5);

// 捕获外部变量
int factor = 2;
auto doubler = [factor](int x) { return x * factor; };

格式化输出 iomanip

`setw(n)`：设置字段宽度；`left/right`：对齐方向；`fixed`：固定小数；`setprecision(n)`：精度；`setfill(c)`：填充字符。
cout << setfill('0') << setw(4) << 42;  // 输出 0042
