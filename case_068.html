案例 68：结构化绑定 structured binding

使用 C++17 结构化绑定简化 tuple、pair、struct 的元素访问。

核心知识点：结构化绑定、auto [a,b]、tuple、map 遍历

完整代码（含逐行注释）

// 案例 68：结构化绑定
// C++17 结构化绑定简化多值返回和 map 遍历
#include <iostream>
#include <tuple>
#include <map>
#include <string>
#include <vector>
using namespace std;

// 返回多个值
tuple<string, int, double> getStudentInfo() {
    return {"Alice", 25, 95.5};
}

struct Point {
    double x, y, z;
};

int main() {
    // 1. 绑定元组
    auto [name, age, score] = getStudentInfo();
    cout << "姓名: " << name << ", 年龄: " << age
         << ", 分数: " << score << endl;

    // 2. 绑定 pair（map 遍历）
    map<string, int> scores = {
        {"Alice", 95}, {"Bob", 88}, {"Charlie", 92}
    };
    cout << "\n成绩表:" << endl;
    for (const auto& [student, grade] : scores) {
        cout << "  " << student << ": " << grade << endl;
    }

    // 3. 绑定结构体成员
    Point p{1.0, 2.0, 3.0};
    auto [x, y, z] = p;
    cout << "\n坐标: (" << x << ", " << y << ", " << z << ")" << endl;

    // 4. 绑定引用以修改原始值
    Point origin{0, 0, 0};
    auto& [ox, oy, oz] = origin;
    ox = 5.0; // 修改 origin.x
    cout << "修改后: (" << origin.x << ", " << origin.y << ", "
         << origin.z << ")" << endl;

    // 5. 在范围 for 中的结构化绑定
    vector<pair<string, int>> items = {
        {"apple", 3}, {"banana", 5}, {"orange", 2}
    };
    cout << "\n物品列表:" << endl;
    for (const auto& [item, quantity] : items) {
        cout << "  " << item << " x " << quantity << endl;
    }

    return 0;
}

语法规则：结构化绑定语法

auto [a, b, c] = someTuple;
auto& [x, y] = somePair;      // 绑定引用
const auto& [k, v] = mapEntry; // const 引用
// 将复合类型的成员分解为独立变量

举一反三：结构化绑定让代码更可读。auto [a,b] 创建独立变量（值拷贝），auto& [a,b] 绑定引用可修改原值。支持 tuple、pair、struct/class（所有非静态 public 成员）。

语法规则：map 遍历最佳实践

map<string, int> m;
for (const auto& [key, value] : m) {
    // key 是 const string&，value 是 const int&
}
// 替代了 auto& kv : m 然后 kv.first/kv.second

举一反三：结构化绑定让 map 遍历语义清晰。注意类型推断：const auto& 使 key 和 value 都是 const 引用（对 map 而言 key 总是 const）。用 auto& 遍历非 const map 时 value 可修改。

语法规则：多返回值惯用法

tuple<T1, T2, T3> func() {
    return {v1, v2, v3};
}
auto [a, b, c] = func();
// 替代 pair 或 out 参数

举一反三：C++17 后多返回值变得优雅。tuple 比 pair 支持更多值。可用 tie 忽略某些返回值（C++17 前）或用 _ 占位（某些编译器扩展）。注意返回值过多时考虑用 struct。
