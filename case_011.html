案例 11：字符串类（手写 String）

完整代码（带逐行注释）

1 #include <iostream> 输入输出流
2 #include <cstring> 包含 strlen、strcpy 等 C 风格字符串函数
3 using namespace std; 使用标准命名空间
4
5 class String { 自定义字符串类的声明
6 private: private 成员只能被类内部的函数访问
7 char* data_; 底层用动态字符数组存储字符串内容
8 size_t size_; size_t 是无符号整数类型，用于表示大小
9
10 public: public 成员可以从外部访问
11 String() : data_(nullptr), size_(0) {} 初始化列表：冒号后面的 data_(nullptr), size_(0)
12
13 String(const char* s) { 构造函数接收一个 C 风格字符串
14 size_ = strlen(s); strlen 计算字符串长度（不包括空字符）
15 data_ = new char[size_ + 1]; new 在堆上分配内存，+1 用于结尾空字符
16 strcpy(data_, s); strcpy 将源字符串复制到目标地址
17 }
18
19 String(const String& other) { 拷贝构造函数（三法则之一）
20 size_ = other.size_; 拷贝大小
21 data_ = new char[size_ + 1]; 分配新内存（深拷贝，不是浅拷贝）
22 strcpy(data_, other.data_); 复制内容
23 }
24
25 String& operator=(const String& other) { 拷贝赋值运算符（三法则之二）
26 if (this == &other) return *this; 自赋值检查：如果是自己赋值给自己，直接返回
27 delete[] data_; 释放旧内存
28 size_ = other.size_;
29 data_ = new char[size_ + 1];
30 strcpy(data_, other.data_);
31 return *this; 返回当前对象的引用，支持链式赋值
32 }
33
34 ~String() { 析构函数（三法则之三），对象销毁时自动调用
35 delete[] data_; 释放动态数组，防止内存泄漏
36 }
37
38 const char* c_str() const { 获取字符串的 C 风格表示
39 return data_; 返回 const char*，外部不能修改
40 }
41
42 size_t length() const { return size_; } 获取长度，内联实现
43
44 String operator+(const String& other) const { + 运算符重载，拼接字符串
45 String result;
46 result.size_ = size_ + other.size_;
47 result.data_ = new char[result.size_ + 1];
48 strcpy(result.data_, data_);
49 strcpy(result.data_ + size_, other.data_); 指针偏移，追加到末尾
50 return result;
51 }
52 }; 类定义结束，必须有分号
53
54 ostream& operator<<(ostream& os, const String& s) { 全局函数重载 << 运算符
55 os << s.c_str();
56 return os;
57 }
58
59 int main() {
60 String s1("Hello"); 调用带参构造函数
61 String s2(" World");
62 String s3 = s1 + s2; 调用 operator+ 和拷贝构造
63
64 cout << "s1 = " << s1 << ", 长度 = " << s1.length() << endl; 测试输出和长度
65 cout << "s2 = " << s2 << ", 长度 = " << s2.length() << endl;
66 cout << "s3 = " << s3 << ", 长度 = " << s3.length() << endl;
67
68 String s4 = s1; 调用拷贝构造函数
69 cout << "s4（拷贝自 s1）= " << s4 << endl;
70 s4 = s2; 调用拷贝赋值运算符
71 cout << "s4（赋值为 s2）= " << s4 << endl;
72 return 0;
73 }

新语法讲解

C++ 三法则 (Rule of Three)

如果一个类定义了析构函数、拷贝构造函数、拷贝赋值运算符中的任意一个，那么通常需要同时定义这三个。因为这三个函数都涉及资源管理（如动态内存）。
// 有了析构函数 ~String()
// 就必须有:
// String(const String& other)    // 拷贝构造
// String& operator=(const String&) // 拷贝赋值

初始化列表 (: member(value))

在构造函数体执行之前执行的初始化语法。对于 const 成员和引用成员，必须用初始化列表。比在函数体赋值更高效。
class Point {
    int x_, y_;
public:
    Point(int x, int y) : x_(x), y_(y) {}
};

运算符重载 (operator)

为自定义类型定义运算符行为。语法：返回类型 operator运算符(参数) { ... }。常用重载：+ - * / = == << >> [ ]
class Complex {
public:
    Complex operator+(const Complex& o) {
        return Complex(real+o.real, imag+o.imag);
    }
};

举一反三

实现 operator==

bool operator==(const String& o) const { return strcmp(data_, o.data_) == 0; }

思路：strcmp 比较两个 C 字符串，相等返回 0

支持移动语义（C++11）

String(String&& o) : data_(o.data_), size_(o.size_) { o.data_ = nullptr; }

思路：移动构造：直接接管资源，避免深拷贝

实现下标访问 [ ]

char& operator[](size_t i) { return data_[i]; }

思路：让 String 可以像数组一样用 s[0] 访问字符
