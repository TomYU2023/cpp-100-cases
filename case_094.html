案例 94：std::mem_fn 成员函数包装

使用 mem_fn 将成员函数包装为可调用对象，用于算法和回调。

核心知识点：mem_fn、成员指针、bind、可调用对象

完整代码（含逐行注释）

// 案例 94：std::mem_fn 成员函数包装
// 将成员函数转为可调用对象
#include <iostream>
#include <functional>
#include <vector>
#include <algorithm>
#include <string>
using namespace std;

class Student {
private:
    string name;
    double score;
public:
    Student(string n, double s) : name(move(n)), score(s) {}

    double getScore() const { return score; }
    const string& getName() const { return name; }
    bool isPass() const { return score >= 60.0; }
    void print() const {
        cout << name << ": " << score << endl;
    }
};

int main() {
    vector<Student> students = {
        {"Alice", 85.5},
        {"Bob", 55.0},
        {"Charlie", 92.0},
        {"David", 58.5},
        {"Eve", 73.0}
    };

    // 1. mem_fn 包装成员函数用于算法
    cout << "=== 所有学生 ===" << endl;
    for_each(students.begin(), students.end(),
             mem_fn(&Student::print));

    // 2. 用 mem_fn 作为排序依据
    sort(students.begin(), students.end(),
         [](const Student& a, const Student& b) {
             return a.getScore() > b.getScore();
         });

    cout << "\n=== 按分数排序 ===" << endl;
    for_each(students.begin(), students.end(),
             mem_fn(&Student::print));

    // 3. 用 mem_fn 做筛选
    auto passCount = count_if(students.begin(), students.end(),
                              mem_fn(&Student::isPass));
    cout << "\n及格人数: " << passCount
         << " / " << students.size() << endl;

    // 4. mem_fn vs lambda vs bind
    // mem_fn: mem_fn(&Student::getScore)
    // lambda: [](const Student& s) { return s.getScore(); }
    // bind:   bind(&Student::getScore, placeholders::_1)

    // 用 mem_fn 基于成员函数返回值查找
    auto it = find_if(students.begin(), students.end(),
                      [](const Student& s) {
                          return s.getScore() > 90.0;
                      });
    if (it != students.end()) {
        cout << "\n第一个超 90 分: " << it->getName()
             << " (" << it->getScore() << ")" << endl;
    }

    return 0;
}

语法规则：std::mem_fn 用法

#include <functional>
auto printFn = mem_fn(&Student::print);
printFn(student); // 等价于 student.print()
// 将成员函数包装为接受对象引用的可调用对象

举一反三：mem_fn 返回一个可调用对象，接受对象指针或引用作为第一个参数。与 bind 的区别：mem_fn 更轻量（无参数绑定开销）。用于将成员函数传给算法。

语法规则：mem_fn 与算法结合

for_each(v.begin(), v.end(), mem_fn(&T::method));
count_if(v.begin(), v.end(), mem_fn(&T::predicate));
transform(v.begin(), v.end(), output, mem_fn(&T::getter));
// C++20 ranges: v | views::transform(mem_fn(&T::getter))

举一反三：mem_fn 将对象方法适配为算法需要的函数形式。比手写 Lambda 更简洁（但 Lambda 更灵活）。适合简单取值/判断的场景。

语法规则：mem_fn vs Lambda vs bind

// mem_fn: 最轻量，只能调用无参成员
// Lambda: 最灵活，可捕获变量，可处理额外参数
// bind: 可绑定部分参数，但语法复杂（C++11/14）
// 推荐: Lambda > mem_fn > bind

举一反三：C++11 后 Lambda 让 bind 基本过时。mem_fn 在简单场景仍有可读性优势。成员函数有参数时 mem_fn 自动适配（第一个参数是对象，后续是方法参数）。
