案例 55：类型萃取 type_traits

使用 type_traits 实现编译期类型判断和条件编译，理解模板元编程基础。

核心知识点：type_traits、enable_if、SFINAE、编译期判断

完整代码（含逐行注释）

// 案例 55：类型萃取 type_traits
// 使用 enable_if 实现编译期条件编译
#include <iostream>
#include <type_traits>
#include <string>
#include <vector>
using namespace std;

// 使用 enable_if 限制模板：仅对整数类型生效
template <typename T>
typename enable_if<is_integral<T>::value, T>::type
add(T a, T b) {
    cout << "整数加法: ";
    return a + b;
}

// 仅对浮点类型生效
template <typename T>
typename enable_if<is_floating_point<T>::value, T>::type
add(T a, T b) {
    cout << "浮点加法: ";
    return a + b;
}

// 类型萃取辅助函数
template <typename T>
void printTypeInfo() {
    cout << "类型: " << typeid(T).name() << endl;
    cout << "  是整数? " << boolalpha << is_integral<T>::value << endl;
    cout << "  是浮点? " << is_floating_point<T>::value << endl;
    cout << "  是指针? " << is_pointer<T>::value << endl;
    cout << "  是 const? " << is_const<T>::value << endl;
    cout << "  sizeof = " << sizeof(T) << endl;
}

// 编译期判断：仅当 T 有 .size() 方法时才调用
template <typename T>
auto getSize(const T& container) -> decltype(container.size()) {
    return container.size();
}

int main() {
    // 根据类型自动选择不同的 add 重载
    cout << add(10, 20) << endl;       // 整数版本
    cout << add(3.14, 2.72) << endl;   // 浮点版本

    cout << "\n=== 类型信息 ===" << endl;
    printTypeInfo<int>();
    cout << "---" << endl;
    printTypeInfo<double>();
    cout << "---" << endl;
    printTypeInfo<string>();

    // 编译期条件判断
    if constexpr (is_integral<int>::value) {
        cout << "\nint 是整数类型，执行整数特有逻辑" << endl;
    }

    vector<int> v = {1, 2, 3, 4, 5};
    cout << "vector size = " << getSize(v) << endl;

    return 0;
}

语法规则：enable_if 条件编译

template<typename T>
typename enable_if<is_integral<T>::value, T>::type
func(T val) { /* 仅整数可用 */ }

举一反三：enable_if 在编译期根据条件「启用」或「禁用」模板。条件为 false 时产生替换失败（SFINAE），模板被排除但不报错。C++17 可用 if constexpr 替代部分场景。

语法规则：type_traits 类型查询

#include <type_traits>
is_integral<T>::value    // 是否整数
is_pointer<T>::value      // 是否指针
is_const<T>::value        // 是否 const
is_same<T, U>::value      // 是否同一类型

举一反三：type_traits 在编译期查询类型属性。_v 后缀别名（C++17）简化写法：is_integral_v 等价于 is_integral::value。广泛用于模板元编程和 SFINAE。

语法规则：if constexpr (C++17)

if constexpr (is_integral<T>::value) {
    // 编译期分支，不满足条件的代码不会被编译
} else {
    // 另一个分支
}

举一反三：if constexpr 比 enable_if 更直观易读。条件在编译期求值，不满足条件的分支完全不会被实例化。适合替代简单的 enable_if 和标签派发。
