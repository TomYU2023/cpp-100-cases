reinterpret_cast 指针类型转换

语法规则：reinterpret_cast 是最底层的类型转换，将指针按位重新解释为目标类型。主要用途：(1) uint8_t* ↔ char* 用于字节级 IO；(2) 整数 ↔ 指针（嵌入式开发）。注意：不改变任何 bit，只是换了个"看法"。
uint64_t x = 0x1234567890ABCDEF;
// 按 char* 逐字节读取
const char* p = reinterpret_cast(&x);
for (int i = 0; i < 8; i++)
    printf("%02x ", (unsigned char)p[i]);
// 小端机器输出: ef cd ab 90 78 56 34 12

ifstream::read 读取指定字节数

语法规则：read(char* s, streamsize n) 从文件中读取恰好 n 字节到缓冲区。不同于 getline，read 不处理换行符，适合二进制数据。读取后可用 gcount() 检查实际读取了多少字节。
uint64_t value;
ifstream f("data.bin", ios::binary);
f.read(reinterpret_cast(&value), sizeof(value));
if (f.gcount() == sizeof(value)) {
    cout << "Read: " << value;
}

  • 压缩级别控制：
    int level = Z_BEST_COMPRESSION;  // 1-9
int ret = ::compress2(dest, &destLen, src, srcLen, level);

    zlib 支持 1（最快）到 9（最优）的压缩级别，以及 Z_DEFAULT_COMPRESSION（默认 6）。compress2 函数可以指定级别。
