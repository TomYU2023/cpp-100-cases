新语法精讲

std::getline 的分隔符参数

语法规则：getline(stream, string, delim) 的第三个参数指定分隔符（默认 \n）。当分隔符不是换行时，getline 一直读到分隔符或 EOF。这在 CSV 解析中极为常用。
istringstream ss("apple,banana,cherry");
string field;
while (getline(ss, field, ',')) {
    cout << field << endl;  // apple / banana / cherry
}
// 不指定分隔符：默认按行
string line;
getline(cin, line);  // 读取一行

std::cerr 标准错误流

语法规则：cerr 输出到标准错误（stderr），与 cout（stdout）独立。cerr 不缓冲、不重定向，错误信息总能被看到。程序运行时可以通过 2> error.log 单独重定向错误输出。
cerr << "Error: file not found" << endl;
cout << "Processing..." << endl;
// 运行时：./a.out 2>/dev/null  # 隐藏错误信息
// 运行时：./a.out > output.txt  # cout 写入文件，cerr 仍显示在终端

二维 vector 的初始化（嵌套初始化列表）

语法规则：C++11 后支持嵌套初始化列表。外层 {} 是 vector 的行列表，内层 {} 是每行的字段列表。编译器自动推导类型。
vector> m = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};
cout << m[1][2];  // 6（第 2 行第 3 列）

举一反三

  • CSV 的引号字段处理：
    if (!field.empty() && field.front() == '"' && field.back() == '"') {
    field = field.substr(1, field.size() - 2);  // 去掉首尾引号
}

    CSV 标准中，如果字段内含逗号或换行，会用引号包裹。生产级 CSV 解析器需要处理引号字段和引号内的转义引号。

  • 使用 tab 分隔的 TSV 格式：
    auto data = readCSV("data.tsv", '\t');
writeCSV("out.tsv", data, '\t');

    TSV（Tab-Separated Values）在某些场景更常见，只需将分隔符改为 \t。
