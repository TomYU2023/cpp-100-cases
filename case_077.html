案例 77：std::distance/advance 迭代器操作

掌握迭代器的距离计算和前进操作，理解各类迭代器的能力差异。

核心知识点：distance、advance、next、prev、迭代器类别

完整代码（含逐行注释）

// 案例 77：std::distance / advance 迭代器操作
// 迭代器的距离计算与跳跃
#include <iostream>
#include <vector>
#include <list>
#include <set>
#include <iterator>
#include <string>
using namespace std;

int main() {
    // 1. distance：计算两个迭代器之间的距离
    vector<int> vec = {10, 20, 30, 40, 50, 60, 70, 80};

    auto start = vec.begin();
    auto target = find(vec.begin(), vec.end(), 60);

    // 距离：从 begin 到找到元素的步数
    ptrdiff_t dist = distance(start, target);
    cout << "60 在第 " << dist << " 个位置（0-based）" << endl;

    // 2. advance：移动迭代器
    auto it = vec.begin();
    advance(it, 3); // 前进 3 步
    cout << "前进 3 步: " << *it << endl; // 40

    advance(it, -2); // 后退 2 步
    cout << "后退 2 步: " << *it << endl; // 20

    // 3. next / prev（不修改原迭代器）
    auto it2 = vec.begin();
    auto nextIt = next(it2, 4); // 返回移动后的新迭代器
    cout << "\nnext(it, 4) = " << *nextIt << endl; // 50
    cout << "原 it 不变: " << *it2 << endl;         // 10

    auto prevIt = prev(vec.end(), 2); // 从 end 前移 2 步
    cout << "prev(end, 2) = " << *prevIt << endl; // 70

    // 4. O(1) vs O(N)：取决于迭代器类别
    cout << "\n=== 复杂度对比 ===" << endl;

    // vector 是随机访问迭代器: distance 和 advance 都是 O(1)
    auto r1 = vec.begin();
    auto r2 = vec.end();
    cout << "vector::distance O(1): " << distance(r1, r2) << endl;

    // list 是双向迭代器: distance 和 advance 都是 O(N)
    list<int> lst = {1, 2, 3, 4, 5};
    auto l1 = lst.begin();
    auto l2 = lst.end();
    cout << "list::distance O(N): " << distance(l1, l2) << endl;

    return 0;
}

语法规则：std::distance

ptrdiff_t n = distance(first, last);
// 返回从 first 到 last 需要的步数
// 随机访问: O(1)，其他: O(N)
// 要求 last 可以从 first 到达（否则 UB）

举一反三：distance 对随机访问迭代器只需指针减法（O(1)），对 list/set 等需遍历计数（O(N)）。返回类型 ptrdiff_t 是有符号整数。first 必须在 last 前面。

语法规则：std::advance

advance(it, n);
// 将 it 前进 n 步（n 可正可负）
// 随机访问: O(1)，其他: O(|n|)
// n 为负数时要求迭代器是双向的

举一反三：advance 相当于 n 次 ++it（或 --it）。对非随机访问迭代器是 O(N)。修改原迭代器。超出范围时行为未定义。配合 distance 可实现二分搜索等算法。

语法规则：std::next / std::prev

auto it2 = next(it, 3);  // it 前进 3 步，返回新迭代器
auto it3 = prev(it, 2);  // it 后退 2 步
default: next(it)  // 前进 1 步
// 不修改原迭代器（与 advance 的区别）

举一反三：next/prev 是 advance 的纯函数版本。更安全（不会无意修改迭代器）。prev 要求双向迭代器。C++11/14 标准函数，需 #include 。
