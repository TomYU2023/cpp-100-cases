案例 1：九九乘法表

使用嵌套 for 循环打印经典的九九乘法表，学习循环控制和格式化输出的基础用法。

核心知识点：嵌套循环、setw 格式化、iomanip

完整代码（含逐行注释）

// 案例 1：九九乘法表
// 使用嵌套 for 循环打印 9x9 乘法表
#include <iostream>
#include <iomanip>   // setw() 用于设置输出宽度
using namespace std;

int main() {
    // 外层循环控制行（乘数 i 从 1 到 9）
    for (int i = 1; i <= 9; i++) {
        // 内层循环控制列（被乘数 j 从 1 到 i）
        for (int j = 1; j <= i; j++) {
            // 打印乘法表达式，setw(2) 确保至少占 2 个字符宽度对齐
            cout << j << "x" << i << "=" << setw(2) << i * j << "  ";
        }
        cout << endl; // 每行结束后换行
    }
    return 0;
}

语法规则：嵌套 for 循环

for (int i = 1; i <= 9; i++) {
    for (int j = 1; j <= i; j++) {
        cout << i * j << " ";
    }
}

举一反三：外层控制行、内层控制列，两层循环各自独立计数。常见用于矩阵遍历、图形打印等场景。

语法规则：setw() 格式化输出

#include <iomanip>
cout << setw(4) << 42;

举一反三： 还提供 setprecision、setfill 等操作符。setw 只对紧接着的一次输出生效。

语法规则：using namespace std;

using namespace std;
// 之后可直接使用 cout、string、vector 等
cout << "Hello" << endl;

举一反三：小型程序常用，大型项目建议用 std::cout 避免命名冲突。头文件中绝不使用 using namespace。
