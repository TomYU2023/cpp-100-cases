3 using namespace std;
4
5 class Shape { 抽象基类
6 public:
7 virtual double area() const = 0; 纯虚函数：子类必须实现
8 virtual void display() const { 虚函数，子类可以覆盖

10 }

14 class Circle : public Shape { 公开继承，Circle 是一个 Shape

17 public:

21 }

新语法讲解

纯虚函数 (= 0) 与抽象类

格式：virtual 返回类型 函数名(参数) const = 0;。定义了纯虚函数的类叫抽象类，不能直接实例化。子类必须实现所有纯虚函数才能实例化。
class Animal {
    virtual void speak() = 0; // 纯虚
};
// Animal a; // 编译错误
class Dog : public Animal {
    void speak() override { cout<<"Woof"; }
};
Dog d; // OK

override 关键字

C++11 引入，写在虚函数声明末尾。编译器会检查该函数确实覆盖了基类虚函数。如果签名不匹配，编译报错，防止拼写错误。
class Base { virtual void foo(int); };
class Derived : public Base {
    void foo(int) override;  // 正确
    // void foo(float) override; // 编译错误
};

虚析构函数

当通过基类指针删除派生类对象时，必须将基类析构函数声明为 virtual，否则派生类析构不会被调用，导致资源泄漏。
Base* p = new Derived();
delete p; // 如果 ~Base() 不是 virtual，~Derived() 不会调用
