复制代码

新语法讲解

transform 数据映射

一元版本：对每个元素应用函数，结果写入输出迭代器。输入和输出可以是同一容器（就地变换）。二元版本：两个输入容器的对应元素一起运算。时间复杂度 O(n)。

举一反三

// 一元: f(x) -> y
transform(in.begin(),in.end(),out.begin(),f);
// 二元: f(x1,x2) -> y
transform(a.begin(),a.end(),b.begin(),out.begin(),f);
// 输出必须足够大！用 back_inserter 可自动扩容

toupper 的 unsigned char 陷阱

toupper 的参数类型是 int，传 char 可能导致符号扩展问题（负数扩展为很大的正数）。需要先转为 unsigned char：`[](unsigned char c){ return toupper(c); }`。

举一反三

// 错误写法
[](char c){ return toupper(c); }
// 正确写法
[](unsigned char c){ return toupper(c); }
// static_cast(c) 等价
