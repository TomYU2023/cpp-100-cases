新语法讲解

map 底层结构

std::map 基于红黑树实现，插入/查找/删除的时间复杂度都是 O(log n)。元素自动按 key 排序。如果需要 O(1) 平均查找，用 std::unordered_map（基于哈希表）。

举一反三

// 选择指南
map m;       // 需要排序时用
unordered_map um; // 追求速度用
// map 内存更紧凑，遍历有序
// unordered_map 查找更快，但内存开销大

结构化绑定 (C++17)

auto [a, b] = pair 可以把 pair/tuple 的成员拆解到单独变量中。遍历 map 时特别方便，直接拿到 key 和 value。

举一反三

pair p = {42, "hello"};
auto [num, text] = p; // num=42, text="hello"
