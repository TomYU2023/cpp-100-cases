案例 22：map 电话号码簿

使用 std::map 实现一个电话号码簿，支持添加、查找、删除、列表功能，练习关联容器的使用。

核心知识点：map 增删查、迭代器、结构化绑定

完整代码（含逐行注释）

// 案例 22：map 电话号码簿
// 使用 std::map 管理姓名-电话映射
#include <iostream>
#include <map>
#include <string>
using namespace std;

class PhoneBook {
private:
    map<string, string> book; // 姓名 -> 电话号码

public:
    // 添加或更新联系人
    void add(const string& name, const string& phone) {
        auto result = book.insert({name, phone});
        if (!result.second) {
            // 已存在则更新
            book[name] = phone;
            cout << "已更新 " << name << endl;
        } else {
            cout << "已添加 " << name << endl;
        }
    }

    // 查找联系人
    void find(const string& name) const {
        auto it = book.find(name);
        if (it != book.end()) {
            cout << name << " -> " << it->second << endl;
        } else {
            cout << "未找到 " << name << endl;
        }
    }

    // 删除联系人
    void remove(const string& name) {
        if (book.erase(name)) {
            cout << "已删除 " << name << endl;
        } else {
            cout << name << " 不存在" << endl;
        }
    }

    // 列出所有联系人（map 自动按 key 排序）
    void listAll() const {
        if (book.empty()) {
            cout << "电话簿为空" << endl;
            return;
        }
        cout << "=== 电话簿（按姓名排序）===" << endl;
        for (const auto& [name, phone] : book) {
            cout << "  " << name << " : " << phone << endl;
        }
    }
};

int main() {
    PhoneBook pb;

    pb.add("张三", "13800138000");
    pb.add("李四", "13900139000");
    pb.add("王五", "13700137000");
    pb.add("张三", "13600136000"); // 重复姓名，更新号码

    pb.find("李四");
    pb.find("赵六");

    pb.listAll();

    pb.remove("王五");
    pb.listAll();

    return 0;
}

语法规则：map::insert 返回值

auto result = book.insert({key, value});
// result.first: 迭代器
// result.second: bool(是否插入成功)

举一反三：insert 在 key 已存在时不覆盖，返回 pair。book[key]=value 则会覆盖已有值。两者结合使用可检测重复。

语法规则：结构化绑定 (C++17)

for (const auto& [name, phone] : book) {
    cout << name << " : " << phone << endl;
}

举一反三：结构化绑定将 pair/tuple/struct 成员直接解构到变量中。也可用于数组：auto [a,b,c] = arr;。map 迭代解构的是 pair。

语法规则：map 与 unordered_map 区别

// map: 红黑树，O(log n)，有序
// unordered_map: 哈希表，O(1) 平均，无序

举一反三：需要有序遍历时用 map，只需快速查找用 unordered_map。unordered_map 需提供自定义类型的哈希函数。
