案例 89：LSM-Tree 键值数据库

实现 memtable（内存红黑树）+ SSTable（磁盘有序文件）+ 压缩合并的简化版 LSM-Tree，理解 LevelDB 的核心思想。

核心知识点：LSM-Tree 架构、lower_bound 二分查找、结构化绑定、多路归并压缩、文件系统操作

完整代码（含逐行注释）

// 案例 89：LSM-Tree 键值数据库
// LSM-Tree（Log-Structured Merge-Tree）是 LevelDB/RocksDB 的核心数据结构。
// 本案例实现 memtable + SSTable + 合并压缩的简化版。

#include <iostream>
#include <map>
#include <vector>
#include <fstream>
#include <sstream>
#include <string>
#include <algorithm>
#include <filesystem>

namespace fs = std::filesystem;

// ─── 常量 ────────────────────────────────────────
const size_t MEMTABLE_LIMIT = 5;  // memtable 达到 5 条就落盘
const std::string DATA_DIR = "lsm_data/";
int sstableSeq = 0;  // SSTable 序号

// ─── SSTable（磁盘上的有序文件）─────────────────
struct SSTable {
    std::string filename;
    std::vector<std::pair<std::string, std::string>> entries;
};

// ─── Memtable（内存红黑树）───────────────────────
std::map<std::string, std::string> memtable;
std::vector<SSTable> sstables;

// ─── 初始化 ──────────────────────────────────────
void init() {
    fs::create_directories(DATA_DIR);
    // 加载已有的 SSTable
    for (const auto& entry : fs::directory_iterator(DATA_DIR)) {
        if (entry.is_regular_file()) {
            SSTable sst;
            sst.filename = entry.path().string();
            std::ifstream in(sst.filename);
            std::string line;
            while (std::getline(in, line)) {
                auto sep = line.find('\t');
                if (sep != std::string::npos)
                    sst.entries.push_back({
                        line.substr(0, sep),
                        line.substr(sep + 1)
                    });
            }
            sstables.push_back(sst);
        }
    }
    // 恢复序号
    if (!sstables.empty())
        sstableSeq = sstables.size();
}

// ─── 写入 ────────────────────────────────────────
void put(const std::string& key, const std::string& value) {
    memtable[key] = value;  // map::operator[] 会覆盖已有 key
    if (memtable.size() >= MEMTABLE_LIMIT)
        flushMemtable();
}

// ─── Memtable 刷盘 ──────────────────────────────
void flushMemtable() {
    std::string fname = DATA_DIR + "sst_" + std::to_string(sstableSeq++) + ".txt";
    std::ofstream out(fname);
    for (const auto& [k, v] : memtable)
        out << k << "\t" << v << "\n";
    out.close();

    // 记录 SSTable
    SSTable sst;
    sst.filename = fname;
    sst.entries.assign(memtable.begin(), memtable.end());
    sstables.push_back(sst);

    std::cout << "[Flush] " << memtable.size() << " 条写入 " << fname << std::endl;
    memtable.clear();
}

// ─── 读取 ────────────────────────────────────────
std::string get(const std::string& key) {
    // 1. 先查 memtable（最新数据）
    auto mit = memtable.find(key);
    if (mit != memtable.end()) return mit->second;

    // 2. 从最新到最旧遍历 SSTable
    for (int i = sstables.size() - 1; i >= 0; i--) {
        // 二分查找（SSTable 已排序）
        auto& entries = sstables[i].entries;
        auto it = std::lower_bound(entries.begin(), entries.end(),
            std::make_pair(key, std::string("")),
            [](const auto& a, const auto& b) { return a.first < b.first; });
        if (it != entries.end() && it->first == key)
            return it->second;
    }
    return "（未找到）";
}

// ─── 压缩合并：将多个 SSTable 合并为一个 ────────
void compact() {
    if (sstables.size() < 2) return;
    std::cout << "[Compact] 合并 " << sstables.size() << " 个 SSTable..." << std::endl;

    // 多路归并：将所有 SSTable 的键值对合并到一个 map（自动去重排序）
    std::map<std::string, std::string> merged;
    for (auto& sst : sstables)
        for (auto& [k, v] : sst.entries)
            merged[k] = v;

    // 删除旧文件
    for (auto& sst : sstables)
        fs::remove(sst.filename);
    sstables.clear();

    // 写回一个 SSTable
    std::string fname = DATA_DIR + "sst_" + std::to_string(sstableSeq++) + ".txt";
    std::ofstream out(fname);
    for (const auto& [k, v] : merged)
        out << k << "\t" << v << "\n";
    out.close();

    SSTable sst;
    sst.filename = fname;
    sst.entries.assign(merged.begin(), merged.end());
    sstables.push_back(sst);
    std::cout << "[Compact] 完成，共 " << merged.size() << " 键。" << std::endl;
}

// ─── 列出所有键 ──────────────────────────────────
void listAll() {
    std::map<std::string, std::string> all;
    for (auto& sst : sstables)
        for (auto& [k, v] : sst.entries)
            all[k] = v;
    for (auto& [k, v] : memtable)
        all[k] = v;

    for (const auto& [k, v] : all)
        std::cout << k << " => " << v << std::endl;
}

// ─── CLI ─────────────────────────────────────────
int main() {
    init();
    std::cout << "=== LSM-Tree 键值数据库 ===\n";
    std::cout << "命令: put k v | get k | list | compact | flush | quit\n";

    while (true) {
        std::cout << "\n> ";
        std::string cmd;
        std::cin >> cmd;

        if (cmd == "quit") break;
        else if (cmd == "put") {
            std::string k, v;
            std::cin >> k >> v;
            put(k, v);
            std::cout << "OK" << std::endl;
        }
        else if (cmd == "get") {
            std::string k;
            std::cin >> k;
            std::cout << get(k) << std::endl;
        }
        else if (cmd == "list") listAll();
        else if (cmd == "compact") compact();
        else if (cmd == "flush") flushMemtable();
        else std::cout << "未知命令" << std::endl;
    }

    flushMemtable();  // 退出前落盘
    return 0;
}

语法规则：std::lower_bound 二分查找

#include <algorithm>
// 在有序序列中查找第一个 >= target 的位置
auto it = std::lower_bound(v.begin(), v.end(), target);
// 需要自定义比较器时：
auto it = std::lower_bound(v.begin(), v.end(), target,
    [](const auto& a, const auto& b) { return a.first < b.first; });

💡 举一反三：lower_bound 要求序列已排序（对 vector 用 std::sort），返回的是迭代器。检查 it->first == key 确认精确匹配。时间复杂度 O(log n)，是 LSM-Tree 查询 SSTable 的关键。

语法规则：std::map + 结构化绑定（C++17）

#include <map>
std::map<std::string, int> m;
m["a"] = 1;
for (const auto& [key, value] : m)
    std::cout << key << " => " << value << std::endl;

💡 举一反三：结构化绑定（Structured Binding）将 pair/tuple/struct 的成员解包为独立变量。auto& [k, v] 是对引用解包（不拷贝）。map 的迭代器解引用后得到 std::pair，绑定为 [key, value] 就是取 pair 的两个成员。

语法规则：filesystem::remove / remove_all 删除文件

#include <filesystem>
std::filesystem::remove("file.txt");        // 删除文件或空目录
std::filesystem::remove_all("dir/");     // 递归删除目录（含内容）

💡 举一反三：remove 对不存在文件抛异常或返回 false（取决于实现）。remove_all 返回删除的文件数。Compaction 阶段需要覆盖写入，先删除旧 SSTable 再生成新文件，保证原子性。
