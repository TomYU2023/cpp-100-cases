案例 48：单例模式(线程安全)

实现一个线程安全的单例模式，使用 C++11 的静态局部变量保证安全。

核心知识点：单例模式、静态局部变量、线程安全

完整代码（含逐行注释）

// 案例 48：单例模式（线程安全）
// C++11 静态局部变量保证线程安全的初始化
#include <iostream>
#include <string>
#include <thread>
#include <vector>
using namespace std;

// 数据库连接管理器（单例）
class DatabaseManager {
private:
    string connectionString;
    bool connected;

    // 私有构造函数
    DatabaseManager() : connectionString("localhost:3306"), connected(false) {
        cout << "DatabaseManager 构造" << endl;
    }

    // 禁止拷贝和赋值
    DatabaseManager(const DatabaseManager&) = delete;
    DatabaseManager& operator=(const DatabaseManager&) = delete;

public:
    // 获取单例实例（C++11 保证线程安全）
    static DatabaseManager& getInstance() {
        // 静态局部变量的初始化在 C++11+ 是线程安全的
        static DatabaseManager instance;
        return instance;
    }

    void connect() {
        if (!connected) {
            // 模拟连接
            cout << "连接数据库: " << connectionString << endl;
            connected = true;
        }
    }

    void query(const string& sql) {
        if (!connected) connect();
        cout << "执行: " << sql << endl;
    }

    ~DatabaseManager() {
        cout << "DatabaseManager 析构" << endl;
    }
};

// 配置管理器（另一个单例）
class ConfigManager {
private:
    ConfigManager() { cout << "ConfigManager 构造" << endl; }
    ConfigManager(const ConfigManager&) = delete;
    ConfigManager& operator=(const ConfigManager&) = delete;

public:
    static ConfigManager& getInstance() {
        static ConfigManager instance;
        return instance;
    }

    string getValue(const string& key) {
        return "value_of_" + key;
    }
};

int main() {
    cout << "=== 线程安全单例 ===" << endl;

    // 多线程同时获取单例
    vector<thread> threads;
    for (int i = 0; i < 5; i++) {
        threads.emplace_back([i]() {
            auto& db = DatabaseManager::getInstance();
            db.query("SELECT * FROM users WHERE id=" + to_string(i));
        });
    }
    for (auto& t : threads) t.join();

    // 验证：多次获取返回同一个实例
    auto& db1 = DatabaseManager::getInstance();
    auto& db2 = DatabaseManager::getInstance();
    cout << "&db1 == &db2: "
         << (&db1 == &db2 ? "true (同一实例)" : "false") << endl;

    return 0;
}

语法规则：C++11 静态局部变量线程安全

static Singleton& getInstance() {
    static Singleton instance; // C++11 保证仅初始化一次，线程安全
    return instance;
}

举一反三：这是 C++ 中最简洁且线程安全的单例实现（Meyers Singleton）。C++11 前需要双重检查锁（DCLP）。不要忘记 delete 拷贝构造和拷贝赋值。

语法规则：= delete 禁止拷贝

DatabaseManager(const DatabaseManager&) = delete;
DatabaseManager& operator=(const DatabaseManager&) = delete;
// 明确禁止拷贝，编译器报错而非静默通过

举一反三：C++11 引入 = delete 明确禁用函数。比 C++03 的 private 声明更明确（错误信息在编译时而非链接时）。也可 delete 任意函数以阻止特定重载。

语法规则：多线程中共享单例

// 多个线程均可通过 getInstance() 获取同一实例
// 静态局部变量初始化是线程安全的
// 但实例方法的并发访问仍需我们自己处理（如需要）

举一反三：单例的「创建」是线程安全的，但「使用」不是。如果单例持有可变状态且被多线程访问，需要在成员函数内部加锁。考虑是否真的需要单例——它本质上是全局状态。
