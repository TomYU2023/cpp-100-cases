28 line = trim(line);
29

新语法精讲

find_if_not 与 Lambda 查找

语法规则：find_if_not(first, last, pred) 返回第一个使 pred 返回 false 的元素。配合 ::isspace 函数指针可找到第一个非空白字符。rbegin/rend 提供反向迭代器，用于从末尾向前搜索。
string s = "  hello  ";
// 正向找第一个非空白
auto start = find_if_not(s.begin(), s.end(), ::isspace);
// 反向找最后一个非空白（注意 base() 转换）
auto end = find_if_not(s.rbegin(), s.rend(), ::isspace).base();
string trimmed(start, end);  // "hello"

stoi / stod / stoll 字符串转数值

语法规则：stoi（string to int）、stod（to double）、stoll（to long long）。可传第二参数（size_t*）获知转换了几个字符，第三参数为基数（默认 10）。无效输入会抛 invalid_argument 或 out_of_range。
int a = stoi("42");           // 42
int b = stoi("0xFF", nullptr, 16); // 255（十六进制）
double c = stod("3.14");          // 3.14
// size_t pos;
// int d = stoi("42abc", &pos);   // d=42, pos=2

举一反三

  • 完整的 ini 写入功能：
    void save(const string& path) {
    ofstream f(path);
    for (auto& [sec, kv] : sections) {
        f << "[" << sec << "]\n";
        for (auto& [k, v] : kv) f << k << " = " << v << "\n";
        f << "\n";
    }
}

    INi 写入只需按照标准格式逐节输出，保持可读性即可。注意空行分隔不同节。

  • 使用 toml11 / inih 等成熟库：
    #include 
auto data = toml::parse("config.toml");
int port = toml::find(data, "server", "port");

    INI 格式简单，但 TOML/YAML/JSON 等更现代的配置格式表达能力更强。生产项目建议用成熟库处理配置文件。
