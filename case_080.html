案例 80：std::set_union/intersection 集合操作

使用标准库的集合算法进行交、并、差运算。

核心知识点：set_union、set_intersection、set_difference、有序集合

完整代码（含逐行注释）

// 案例 80：std::set_union / intersection 集合操作
// 有序集合的交、并、差运算
#include <iostream>
#include <algorithm>
#include <vector>
#include <set>
#include <string>
#include <iterator>
using namespace std;

int main() {
    // 准备两个有序集合
    vector<int> A = {1, 2, 3, 4, 5, 6, 7};
    vector<int> B = {5, 6, 7, 8, 9, 10};
    vector<int> result;

    // 1. 并集 Union
    cout << "A: ";
    for (int x : A) cout << x << " ";
    cout << "\nB: ";
    for (int x : B) cout << x << " ";
    cout << endl;

    result.clear();
    set_union(A.begin(), A.end(), B.begin(), B.end(),
              back_inserter(result));
    cout << "\n并集 (A ∪ B): ";
    for (int x : result) cout << x << " ";
    cout << endl;

    // 2. 交集 Intersection
    result.clear();
    set_intersection(A.begin(), A.end(), B.begin(), B.end(),
                     back_inserter(result));
    cout << "交集 (A ∩ B): ";
    for (int x : result) cout << x << " ";
    cout << endl;

    // 3. 差集 Difference
    result.clear();
    set_difference(A.begin(), A.end(), B.begin(), B.end(),
                   back_inserter(result));
    cout << "差集 (A - B): ";
    for (int x : result) cout << x << " ";
    cout << endl;

    result.clear();
    set_difference(B.begin(), B.end(), A.begin(), A.end(),
                   back_inserter(result));
    cout << "差集 (B - A): ";
    for (int x : result) cout << x << " ";
    cout << endl;

    // 4. 对称差集
    result.clear();
    set_symmetric_difference(A.begin(), A.end(), B.begin(), B.end(),
                             back_inserter(result));
    cout << "对称差集 (A Δ B): ";
    for (int x : result) cout << x << " ";
    cout << endl;

    // 5. includes: 判断子集关系
    cout << "\nA ⊆ A ? " << boolalpha
         << includes(A.begin(), A.end(), A.begin(), A.end()) << endl;

    return 0;
}

语法规则：std::set_union

set_union(first1, last1, first2, last2, output);
// 求两个有序范围的并集
// 要求输入已排序，输出也排序
// O(N+M) 时间复杂度

举一反三：所有集合算法都要求输入已排序（升序）。不排序的结果不符合「集合」语义。输出用 back_inserter 自动扩张。可用自定义比较器（如降序）。

语法规则：std::set_intersection

set_intersection(A.begin(), A.end(),
    B.begin(), B.end(), back_inserter(result));
// 找出同时存在于两个集合中的元素
// 重复元素按 min(count_A, count_B) 保留

举一反三：交集是最常用的集合操作。对于 multiset（重复元素），保留每个元素在两个集合中出现次数的较小值。适合查找共同用户、共同标签等。

语法规则：std::includes

bool sub = includes(A.begin(), A.end(),
    B.begin(), B.end());
// 检查 B 是否是 A 的子集
// 等价于 set_difference 后的结果为空

举一反三：includes 返回 bool，比 set_difference + empty 更高效（可能提前终止）。注意：子集定义包含相等的情况（A ⊆ A 返回 true）。同样要求有序输入。
