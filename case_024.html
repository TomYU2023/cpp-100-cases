案例 24：set 集合运算

使用 std::set 实现集合的交、并、差运算，练习 set 的特性和 STL 集合算法。

核心知识点：set 特性、set_union、set_intersection、set_difference

完整代码（含逐行注释）

// 案例 24：set 集合运算
// 使用 STL 算法实现集合交、并、差
#include <iostream>
#include <set>
#include <vector>
#include <algorithm>
#include <iterator>
using namespace std;

// 打印 set
void printSet(const string& label, const set<int>& s) {
    cout << label << " = {";
    for (auto it = s.begin(); it != s.end(); ++it) {
        if (it != s.begin()) cout << ", ";
        cout << *it;
    }
    cout << "}" << endl;
}

int main() {
    set<int> setA = {1, 3, 5, 7, 9};
    set<int> setB = {3, 6, 7, 9, 12};

    printSet("A", setA);
    printSet("B", setB);

    // 1. 并集：A ∪ B
    set<int> unionSet;
    set_union(setA.begin(), setA.end(),
              setB.begin(), setB.end(),
              inserter(unionSet, unionSet.begin()));
    printSet("A ∪ B", unionSet);

    // 2. 交集：A ∩ B
    set<int> interSet;
    set_intersection(setA.begin(), setA.end(),
                     setB.begin(), setB.end(),
                     inserter(interSet, interSet.begin()));
    printSet("A ∩ B", interSet);

    // 3. 差集：A - B
    set<int> diffSet;
    set_difference(setA.begin(), setA.end(),
                   setB.begin(), setB.end(),
                   inserter(diffSet, diffSet.begin()));
    printSet("A - B", diffSet);

    // 4. 对称差集：(A ∪ B) - (A ∩ B)
    set<int> symDiffSet;
    set_symmetric_difference(setA.begin(), setA.end(),
                             setB.begin(), setB.end(),
                             inserter(symDiffSet, symDiffSet.begin()));
    printSet("对称差", symDiffSet);

    return 0;
}

语法规则：std::set 有序集合

#include <set>
set<int> s = {3, 1, 2}; // 自动排序为 {1, 2, 3}
s.insert(4);
if (s.count(2)) { /* ... */ }

举一反三：set 基于红黑树，元素唯一且自动排序。插入、查找、删除均为 O(log n)。允许多个重复 key 用 multiset。

语法规则：集合运算算法

set_union(a.begin(),a.end(), b.begin(),b.end(), inserter(result, result.begin()));

举一反三：set_union/intersection/difference 要求输入区间有序。inserter 适配器让算法通过 insert() 写入结果（而非覆盖）。时间复杂度 O(n+m)。

语法规则：inserter 迭代器适配器

#include <iterator>
inserter(container, container.begin()); // 通过 insert 插入

举一反三：另有 back_inserter（push_back）和 front_inserter（push_front）。适配器使输出迭代器适配不同容器的插入方式。
