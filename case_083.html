案例 83：std::next_permutation 排列生成

使用 next_permutation 生成全排列，解决排列组合类算法问题。

核心知识点：next_permutation、全排列、字典序、排列遍历

完整代码（含逐行注释）

// 案例 83：std::next_permutation 排列生成
// 生成所有排列组合
#include <iostream>
#include <algorithm>
#include <vector>
#include <string>
using namespace std;

int main() {
    // 1. 基础：生成数字全排列
    vector<int> nums = {1, 2, 3};

    cout << "=== 全排列 (1, 2, 3) ===" << endl;
    int count = 0;
    do {
        cout << ++count << ": ";
        for (int x : nums) cout << x << " ";
        cout << endl;
    } while (next_permutation(nums.begin(), nums.end()));
    cout << "共 " << count << " 种排列" << endl;

    // 2. 必须从有序序列开始才能生成全部
    cout << "\n=== 从非有序开始会漏 ===" << endl;
    vector<int> unsorted = {3, 1, 2};
    count = 0;
    do {
        cout << ++count << ": ";
        for (int x : unsorted) cout << x << " ";
        cout << endl;
    } while (next_permutation(unsorted.begin(), unsorted.end()));
    cout << "仅 " << count << " 种（漏了前面的）" << endl;

    // 3. 字符串排列
    cout << "\n=== 字符串排列 ===" << endl;
    string s = "ABC";
    sort(s.begin(), s.end()); // 先排序
    do {
        cout << s << endl;
    } while (next_permutation(s.begin(), s.end()));

    // 4. prev_permutation：递减排列
    cout << "\n=== 递减排列 ===" << endl;
    vector<int> desc = {4, 3, 2, 1};
    do {
        for (int x : desc) cout << x << " ";
        cout << endl;
    } while (prev_permutation(desc.begin(), desc.end()));

    return 0;
}

语法规则：std::next_permutation

bool hasNext = next_permutation(v.begin(), v.end());
// 生成字典序中的下一个排列
// 返回 true 表示成功，false 表示已是最大排列
// 要求从已排序序列开始才能生成全排列

举一反三：next_permutation 基于字典序。复杂度 O(N)（一次排列）。从 sorted 开始生成 N! 次（对 N≤10 可用）。实际是对数组的原地重排。

语法规则：next vs prev_permutation

// next_permutation: 递增排列（从小到大）
// prev_permutation: 递减排列（从大到小）
// 前者从升序开始，后者从降序开始
// 两者互为逆操作

举一反三：两者返回 false 时排列循环回初始顺序。prev_permutation 适合从大到小遍历。对一个排列连续调用 next+prev 应该还原。

语法规则：排列算法应用

// 1. 旅行商问题 (TSP) 的暴力解
// 2. 密码破解：尝试所有字母组合
// 3. 测试数据生成：覆盖所有输入顺序
// 4. 组合问题：配合 bool 数组实现 C(n,k)

举一反三：next_permutation 对 N≤12 的排列问题可行。N 更大时需要回溯剪枝或动态规划。注意 N! 增长极快（10! = 3.6M, 15! = 1.3T）。
