案例 23：unordered_map LRU缓存

使用 unordered_map + list 实现一个 LRU 缓存，练习数据结构的组合使用。

核心知识点：list、unordered_map、LRU 算法、迭代器失效

完整代码（含逐行注释）

// 案例 23：unordered_map LRU 缓存
// list + unordered_map 实现 O(1) 的 LRU 缓存
#include <iostream>
#include <list>
#include <unordered_map>
using namespace std;

template <typename K, typename V>
class LRUCache {
private:
    size_t capacity;
    // 双向链表存储 (key, value) 对，头部最新，尾部最旧
    list<pair<K, V>> items;
    // key -> 链表迭代器，O(1) 定位
    unordered_map<K, typename list<pair<K, V>>::iterator> index;

public:
    // 构造函数
    explicit LRUCache(size_t cap) : capacity(cap) {}

    // 获取值，存在则移到链表头部
    optional<V> get(const K& key) {
        auto it = index.find(key);
        if (it == index.end()) {
            return nullopt; // 未命中
        }
        // 命中：移到链表头部（splice 是 O(1)）
        items.splice(items.begin(), items, it->second);
        return it->second->second;
    }

    // 放入键值对
    void put(const K& key, const V& value) {
        auto it = index.find(key);

        if (it != index.end()) {
            // key 已存在：更新值并移到头部
            it->second->second = value;
            items.splice(items.begin(), items, it->second);
            return;
        }

        // key 不存在：检查容量
        if (items.size() >= capacity) {
            // 淘汰最旧的（链表尾部）
            auto last = items.back();
            index.erase(last.first);
            items.pop_back();
            cout << "淘汰: " << last.first << endl;
        }

        // 插入到链表头部
        items.emplace_front(key, value);
        index[key] = items.begin();
    }

    // 打印当前缓存内容
    void print() const {
        cout << "缓存内容（新->旧）: ";
        for (const auto& [k, v] : items) {
            cout << k << "=" << v << "  ";
        }
        cout << endl;
    }
};

int main() {
    LRUCache<string, int> cache(3);

    cache.put("A", 1); cache.print();
    cache.put("B", 2); cache.print();
    cache.put("C", 3); cache.print();
    cache.get("A"); // A 被访问，移到头部
    cache.print();
    cache.put("D", 4); // 容量满，淘汰 B
    cache.print();

    return 0;
}

语法规则：list::splice 移动节点

items.splice(items.begin(), items, it); // 将 it 移到头部

举一反三：splice 是 O(1) 操作，直接重连链表指针不拷贝数据。这是 LRU 能实现 O(1) 的关键。list 的迭代器是双向迭代器，不提供随机访问。

语法规则：unordered_map 存迭代器

unordered_map<K, list<pair<K,V>>::iterator> index;
// 直接通过迭代器访问链表元素

举一反三：list 的增删不会导致其他迭代器失效，所以可以安全地在 map 中缓存迭代器。如果换成 vector/deque 则扩容时迭代器会失效。

语法规则：emplace_front 原位构造

items.emplace_front(key, value); // 直接在链表节点上构造 pair
// 优于 push_front(make_pair(key, value)) 减少一次拷贝

举一反三：emplace 系列函数直接在容器内存上构造元素，避免临时对象的创建和拷贝。所有容器都提供了 emplace 系列方法。
