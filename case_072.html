案例 72：std::tuple 元组操作

深入学习 tuple 的创建、访问、拼接和遍历技巧。

核心知识点：tuple、tie、tuple_cat、get、结构化绑定

完整代码（含逐行注释）

// 案例 72：std::tuple 元组操作
// tuple 的创建、解包、拼接和遍历
#include <iostream>
#include <tuple>
#include <string>
#include <type_traits>
using namespace std;

// 打印 tuple 的辅助函数（编译期递归）
template <size_t Index = 0, typename... Types>
void printTuple(const tuple<Types...>& t) {
    if constexpr (Index < sizeof...(Types)) {
        if constexpr (Index > 0) cout << ", ";
        cout << get<Index>(t);
        printTuple<Index + 1>(t);
    }
}

template <typename... Types>
void print(const string& label, const tuple<Types...>& t) {
    cout << label << "(";
    printTuple(t);
    cout << ")" << endl;
}

int main() {
    // 1. 创建 tuple
    auto t1 = make_tuple(42, 3.14, string("hello"));
    tuple<int, string, bool> t2(25, "world", true);

    print("t1", t1);
    print("t2", t2);

    // 2. 访问元素
    cout << "\nt1<0>: " << get<0>(t1) << endl;
    cout << "t1<1>: " << get<1>(t1) << endl;
    cout << "t1<2>: " << get<2>(t1) << endl;

    // 按类型访问（仅当类型唯一时可用）
    cout << "t2中的string: " << get<string>(t2) << endl;

    // 3. tie 解包
    int a;
    double b;
    string c;
    tie(a, b, c) = t1;
    cout << "\ntie 解包: " << a << ", " << b << ", " << c << endl;

    // tie 忽略某些值
    int x;
    tie(x, ignore, ignore) = t1;
    cout << "只取第一个: " << x << endl;

    // 4. tuple_cat 拼接
    auto t3 = tuple_cat(t1, make_tuple('A', 99));
    print("\n拼接后 t3", t3);

    // 5. 获取 tuple 大小
    cout << "\nt1 元素数: " << tuple_size<decltype(t1)>::value << endl;

    // 6. 交换
    tuple<int, int> ta(1, 2), tb(3, 4);
    ta.swap(tb);
    print("交换后 ta", ta);

    return 0;
}

语法规则：tuple 创建与访问

auto t = make_tuple(42, 3.14, "hello");
get<0>(t);         // 按索引访问
get<int>(t);       // 按类型访问（仅唯一类型）
tuple_size<decltype(t)>::value; // 元素数量

举一反三：make_tuple 自动推导类型。get 的索引必须是编译期常量。按类型访问要求 tuple 中该类型只出现一次，否则编译错误。C++14 可用 get 简化。

语法规则：tie 解包与 ignore

int a, b;
tie(a, b) = make_pair(1, 2);
tie(a, ignore) = make_tuple(3, 4.5, 'c');
// ignore 跳过不需要的值
// tie 创建的是左值引用

举一反三：tie 将已有变量绑定到 tuple 元素（引用语义，可修改原 tuple）。ignore 是 std::ignore 的实例。C++17 结构化绑定 auto [a,b] 更简洁且语法更自然。

语法规则：tuple_cat 拼接

auto t3 = tuple_cat(t1, t2, make_tuple(42));
// 将多个 tuple 拼接成一个
// 元素类型和值均保留
// 结果类型是编译期确定的

举一反三：tuple_cat 在编译期将多个 tuple 的成员列表合并。适合需要动态组合返回值的场景。注意：拼接后元素顺序是参数从左到右，各个 tuple 的成员顺序保留。
