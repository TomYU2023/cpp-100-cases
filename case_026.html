案例 26：deque 滑动窗口

使用 deque 实现滑动窗口的最大值查找，练习双端队列的特性应用。

核心知识点：deque 双端操作、单调队列、滑动窗口

完整代码（含逐行注释）

// 案例 26：deque 滑动窗口最大值
// 使用单调递减双端队列，O(n) 找出所有窗口的最大值
#include <iostream>
#include <deque>
#include <vector>
using namespace std;

// 返回每个窗口中的最大值
vector<int> maxSlidingWindow(const vector<int>& nums, int k) {
    vector<int> result;
    // deque 存储下标，保持元素在 deque 中单调递减
    deque<int> dq;

    for (int i = 0; i < (int)nums.size(); i++) {
        // 1. 移除超出窗口左边界的位置
        if (!dq.empty() && dq.front() <= i - k) {
            dq.pop_front();
        }

        // 2. 维护单调递减：比新元素小的都弹出
        while (!dq.empty() && nums[dq.back()] < nums[i]) {
            dq.pop_back();
        }

        // 3. 当前元素入队
        dq.push_back(i);

        // 4. 当窗口形成后（i >= k-1），记录最大值
        if (i >= k - 1) {
            result.push_back(nums[dq.front()]);
        }
    }
    return result;
}

int main() {
    vector<int> nums = {1, 3, -1, -3, 5, 3, 6, 7};
    int k = 3;

    cout << "数组: ";
    for (int n : nums) cout << n << " ";
    cout << endl;

    vector<int> result = maxSlidingWindow(nums, k);

    cout << "滑动窗口大小=" << k << " 的最大值: ";
    for (int n : result) cout << n << " ";
    cout << endl;

    return 0;
}

语法规则：deque 双端操作

#include <deque>
dq.push_front(1);   // 头部插入
dq.push_back(2);    // 尾部插入
dq.pop_front();     // 头部删除
dq.pop_back();      // 尾部删除

举一反三：deque 支持 O(1) 的头尾插入删除。内存分段存储，不像 vector 整体连续。适合需要在两端频繁操作的场景。

语法规则：单调队列技巧

// 维护队列元素单调（递增或递减）
// 入队前将破坏单调性的元素出队
while (!dq.empty() && nums[dq.back()] < nums[i])
    dq.pop_back();

举一反三：单调队列是滑动窗口问题的经典解法。每个元素最多入队出队各一次，总复杂度 O(n)。也可用 priority_queue 但复杂度是 O(n log n)。

语法规则：deque 随机访问

dq[0];    // 随机访问 O(1)
dq.at(3); // 带边界检查

举一反三：deque 支持随机访问但不如 vector 快（需计算块偏移）。当需要头尾插入 + 随机访问时，deque 是最佳选择。
