案例 5：数组去重 + 排序

完整代码（带逐行注释）

1 #include
2 #include // sort(), unique()
3 #include // vector
4 using namespace std;
5
6 int main() {
7 // 初始化一个包含重复元素的数组
8 int arr[] = {3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5};
9 int n = sizeof(arr) / sizeof(arr[0]); 初始化列表 {} 是 C++11 语法，用于初始化数组
10 sizeof(arr) 返回整个数组的字节数，sizeof(arr[0]) 是单个元素的字节数，相除得到元素个数
11 cout << "原始数组：";
12 for (int i = 0; i < n; i++) cout << arr[i] << " ";
13 arr[i] 通过下标访问第 i 个元素（从 0 开始）
14 // 第一步：排序（sort 底层是快速排序，O(n log n)）
15 sort(arr, arr + n);
16 sort(起始指针, 结束指针) — 数组名退化为首元素指针，arr+n 是 past-the-end 指针
17 cout << "\n排序后：";
18 for (int i = 0; i < n; i++) cout << arr[i] << " ";
19
20 // 第二步：去重（unique 把重复元素移到末尾，返回新结尾的迭代器）
21 int* newEnd = unique(arr, arr + n); unique 返回一个指针，指向去重后逻辑结尾的下一个位置
22 // newEnd 指向去重后数组的"逻辑末尾"
23 int newSize = newEnd - arr; 指针相减得到两个地址之间的元素个数
24
25 cout << "\n去重后：";
26 for (int i = 0; i < newSize; i++) cout << arr[i] << " ";
27 cout << endl;
28
29 // ====== 用 vector 实现的版本（更现代） ====== vector 是可动态增长的数组容器
30 vector vec = {3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5}; vec.begin() 返回指向第一个元素的迭代器，vec.end() 指向末尾之后
31 sort(vec.begin(), vec.end());
32 // erase + unique 组合拳：真正删除重复元素 erase(first, last) 删除 [first, last) 范围内的元素
33 vec.erase(unique(vec.begin(), vec.end()), vec.end());
34
35 cout << "vector 版去重排序："; for (int x : vec) 是范围 for 循环，遍历容器中每个元素（C++11）
36 for (int x : vec) cout << x << " "; // 范围 for 循环（C++11）
37 cout << endl;
38
39 return 0;
40 }

新语法讲解

数组与指针的关系

数组名在表达式中会退化为指向首元素的指针。`arr` 和 `&arr[0]` 等价。`arr + i` 等价于 `&arr[i]`。
int a[5] = {1,2,3,4,5};
int* p = a;       // p 指向 a[0]
cout << *(p+2);   // 输出 3

迭代器 (iterator)

迭代器是泛化的指针，用于遍历容器。`vec.begin()` 返回指向首元素的迭代器，`vec.end()` 返回指向末尾之后。
vector v = {1,2,3};
for (auto it = v.begin(); it != v.end(); ++it)
    cout << *it << " ";

范围 for 循环 (range-based for)

`for (类型 变量 : 容器)` 自动遍历容器中的每个元素。C++11 引入，是最简洁的遍历方式。
vector names = {"Alice","Bob"};
for (const string& s : names) cout << s;
