案例 5：数组去重+排序

对一个整数数组进行去重并排序输出，练习数组操作、STL 容器的基本使用。

核心知识点：vector、sort、unique、迭代器

完整代码（含逐行注释）

// 案例 5：数组去重 + 排序
// 使用 STL 算法高效完成去重和排序
#include <iostream>
#include <vector>    // 动态数组
#include <algorithm> // sort()、unique()
using namespace std;

int main() {
    // 初始化一个包含重复元素的数组
    vector<int> nums = {5, 2, 8, 2, 9, 1, 5, 5, 8, 3};

    cout << "原始数组：";
    for (int n : nums) cout << n << " ";
    cout << endl;

    // 第一步：排序（unique 只能移除相邻的重复元素，所以必须先排序）
    sort(nums.begin(), nums.end());

    // 第二步：去重，unique 返回新逻辑结尾的迭代器
    auto last = unique(nums.begin(), nums.end());

    // 第三步：删除尾部无用元素
    nums.erase(last, nums.end());

    cout << "去重排序后：";
    for (int n : nums) cout << n << " ";
    cout << endl;

    return 0;
}

语法规则：vector 动态数组

#include <vector>
vector<int> v = {1, 2, 3};
v.push_back(4);  // 末尾追加
cout << v.size(); // 获取大小

举一反三：vector 是 C++ 最常用的容器，连续内存存储，支持随机访问。相比 C 数组无需手动管理内存。

语法规则：sort + unique 组合去重

sort(v.begin(), v.end());
auto last = unique(v.begin(), v.end());
v.erase(last, v.end());

举一反三：这是 C++ 经典去重模式。unique 只移动元素不删除，必须配合 erase 才能真正删除尾部重复项。

语法规则：范围 for 循环

for (int n : nums) {
    cout << n << " ";
}

举一反三：C++11 引入，简洁遍历容器。如需修改元素使用 for (int &n : nums)。只读场景用 for (const int &n : nums) 避免拷贝。
