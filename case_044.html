新语法精讲

std::fill 填充算法

语法规则：std::fill(first, last, value) 将 [first, last) 范围内的所有元素设为 value。原理是迭代器循环赋值，比手写 for 循环更清晰且编译器可能优化为 memset。
int arr[10];
fill(arr, arr + 10, 42);  // arr 全部填为 42
vector v(100);
fill(v.begin(), v.end(), -1);  // 全填 -1

位运算提取 6 位组

语法规则：Base64 编码的核心是将 24 位（3 字节）分成 4 个 6 位组。提取第 k 个 6 位组：(triple >> (18 - k*6)) & 0x3F。右移去掉右侧比特，& 0x3F（二进制 111111）掩码取出低 6 位。
uint32_t x = 0b101010101010101010101010; // 24 位
int g0 = (x >> 18) & 0x3F; // 取最高 6 位
int g1 = (x >> 12) & 0x3F; // 取次高 6 位
int g2 = (x >> 6)  & 0x3F; // 取次低 6 位
int g3 =  x        & 0x3F; // 取最低 6 位

fill 与 memset 的区别

语法规则：fill 是 C++ 算法，适用于所有迭代器类型的容器，类型安全。memset 是 C 函数，只能按字节填充原始内存，速度快但仅适用于 POD 类型。
// fill: 类型安全，适用范围广
fill(v.begin(), v.end(), MyClass{42}); // OK
// memset: 仅适用于原始内存
int arr[100];
memset(arr, 0, sizeof(arr));  // OK
// memset(arr, 42, size); // 危险！每字节都会是 42
