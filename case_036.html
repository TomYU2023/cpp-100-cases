新语法讲解

regex_match vs regex_search

regex_match：整个字符串必须完全匹配正则。regex_search：搜索字符串中任意位置的匹配。regex_replace：替换匹配的部分。三者各有用途，不要混用。

举一反三

string s = "abc123def";
regex r(R"(\d+)");
regex_match(s, r);  // false（不含开头的 abc）
regex_search(s, r); // true（找到 123）

R"(...)" 原始字符串字面量

R"(delimiter)...) 允许字符串中包含任意字符而不需要转义。在写正则表达式时特别有用，因为正则会大量使用反斜杠。定界符可以是 "(...)" 或自定义 R"xxx(...)xxx"。

举一反三

string s = "C:\\Users\\name"; // 传统写法
string s = R"(C:\Users\name)";   // 原始字符串
// 如果字符串中有 )" ，可以用自定义定界符
string s = R"delim(...contains )"... )delim";
