案例 20：variant + optional

使用 C++17 的 std::variant 和 std::optional 实现安全的多类型返回和可选值处理。

核心知识点：variant、optional、visit、值语义

完整代码（含逐行注释）

// 案例 20：variant + optional
// C++17 类型安全的多类型联合和可选值
#include <iostream>
#include <variant>   // variant
#include <optional>  // optional
#include <string>
#include <map>
using namespace std;

// 函数可返回 int、double 或 string 三种类型之一
variant<int, double, string> parseValue(const string& input) {
    try {
        // 尝试转为 int
        size_t pos;
        int i = stoi(input, &pos);
        if (pos == input.size()) return i;
    } catch (...) {}

    try {
        // 尝试转为 double
        size_t pos;
        double d = stod(input, &pos);
        if (pos == input.size()) return d;
    } catch (...) {}

    // 最后当作字符串
    return input;
}

// 使用 optional 的安全查找
optional<string> findUser(const map<int, string>& users, int id) {
    auto it = users.find(id);
    if (it != users.end()) {
        return it->second;         // 找到了
    }
    return nullopt;                 // 没找到
}

int main() {
    // === variant 用法 ===
    vector<string> inputs = {"42", "3.14", "hello"};

    for (const auto& s : inputs) {
        auto val = parseValue(s);
        // visit 模式匹配：根据当前实际类型执行对应 Lambda
        visit([](const auto& v) {
            using T = decay_t<decltype(v)>;
            if constexpr (is_same_v<T, int>)
                cout << "整数: " << v << endl;
            else if constexpr (is_same_v<T, double>)
                cout << "浮点: " << v << endl;
            else
                cout << "字符串: " << v << endl;
        }, val);
    }

    // === optional 用法 ===
    map<int, string> users = {{1, "Alice"}, {2, "Bob"}};

    auto result = findUser(users, 1);
    if (result.has_value()) {          // 或 if (result)
        cout << "找到用户: " << result.value() << endl;
    }

    // 使用 value_or 提供默认值
    auto noResult = findUser(users, 99);
    cout << "用户99: " << noResult.value_or("未知用户") << endl;

    return 0;
}

语法规则：std::variant 多类型联合

#include <variant>
variant<int, double, string> v;
v = 42;   // 存 int
v = 3.14; // 存 double

举一反三：variant 是类型安全的 union，自动管理当前存储的类型。用 get() 提取值（类型不对抛 bad_variant_access），或用 get_if() 返回指针。

语法规则：std::visit 模式匹配

visit([](const auto& v) {
    // 对每种可能的类型执行不同逻辑
}, variantValue);

举一反三：visit 是处理 variant 的推荐方式，编译期保证所有类型都被处理。结合 if constexpr 实现类型分派。C++20 支持用 overloaded 模式。

语法规则：std::optional 可选值

#include <optional>
optional<int> find();
if (result) cout << *result;
// value_or(default) 提供默认值

举一反三：optional 代替指针判空或哨兵值（如 -1），语义更明确。has_value() 或隐式 bool 转换检查是否有值。value_or() 安全获取或默认值。
