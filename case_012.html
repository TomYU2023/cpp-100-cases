新语法讲解

const 成员函数

在成员函数声明末尾加 const，表示该函数不修改任何成员变量。const 对象只能调用 const 成员函数。
class Point {
    int x;
public:
    int getX() const { return x; }
    void setX(int v) { x = v; }  // 不能加 const
};
const Point p;
p.getX();  // OK
// p.setX(5); // 错误

友元 (friend)

friend 声明让一个普通函数或另一个类可以访问当前类的 private 和 protected 成员。打破封装性，需谨慎使用。
class A {
    int secret = 42;
    friend void peek(const A& a);
};
void peek(const A& a) { cout << a.secret; } // OK
