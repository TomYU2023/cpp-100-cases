案例 67：std::format 格式化

使用 C++20 format 库进行类型安全的字符串格式化，替代 printf 和 stringstream。

核心知识点：format、格式化占位符、类型安全、编译期检查

完整代码（含逐行注释）

// 案例 67：std::format 格式化
// 类型安全的字符串格式化（C++20）
#include <iostream>
#include <format>
#include <string>
#include <vector>
#include <chrono>
using namespace std;

int main() {
    // 1. 基本格式化
    cout << format("Hello, {}!", "World") << endl;
    cout << format("整数: {}, 浮点数: {:.2f}", 42, 3.14159) << endl;

    // 2. 位置参数
    cout << format("{1} 在 {0} 之前", " 」之后", " 」之前") << endl;

    // 3. 对齐、宽度、填充
    cout << format("|{:>10}|", "右对齐") << endl;   // 右对齐 10 字符宽
    cout << format("|{:<10}|", "左对齐") << endl;   // 左对齐
    cout << format("|{:^10}|", "居中") << endl;     // 居中
    cout << format("|{:*^10}|", "星号填充") << endl; // * 填充

    // 4. 进制和精度
    cout << format("十六进制: {:#x}, 八进制: {:#o}, 二进制: {:#b}",
                   255, 255, 255) << endl;
    cout << format("科学计数: {:.2e}", 12345.6789) << endl;
    cout << format("百分比: {:.1%}", 0.856) << endl;

    // 5. 格式化为字符串
    string msg = format("结果: x={}, y={:.2f}, name=\"{}\"",
                        10, 3.14159, "test");
    cout << msg << endl;

    // 6. 表格对齐
    cout << "\n=== 数据表格 ===" << endl;
    cout << format("{:<8} {:<6} {:>8}\n", "姓名", " 」年龄", " 」分数");
    cout << format("{:-<8} {:-<6} {:->8}\n", "", "", "");
    cout << format("{:<8} {:<6} {:>8.1f}\n", "Alice", 25, 95.5);
    cout << format("{:<8} {:<6} {:>8.1f}\n", "Bob", 30, 88.0);

    return 0;
}

语法规则：format 基本占位符

format("{} + {} = {}", a, b, a + b);
// {} 自动按顺序匹配参数
// {0} {1} 指定参数索引
// {:.2f} 格式说明（2 位小数）

举一反三：format 是类型安全的（编译期检查参数类型），比 printf 安全，比 stringstream 简洁。{} 占位符不需要 %d %s 等类型标记。不匹配的类型编译报错。

语法规则：格式说明符

{:>10}  右对齐，宽度 10
{:<10}  左对齐
{:^10}  居中
{:*^10} 用 * 填充并居中
{:#x}   带 0x 前缀的十六进制
{:.2f}  2 位小数

举一反三：格式说明符是 format 的强大之处。格式：[[fill]align][sign][#][0][width][.precision][type]。类似 Python 的 f-string 和 C# 的 string interpolation。

语法规则：format_to / format_to_n

string buffer;
format_to(back_inserter(buffer), "x={}", x);
// 格式化到已有容器（而非创建新 string）
format_to_n(buf, max, "{}", x); // 限制长度，防止溢出

举一反三：format_to 适合追加到已有字符串或文件（避免多次分配）。format_to_n 限制最大写入字符数，适合固定缓冲区（如日志缓冲区）。C++23 引入 print() 直接输出。
