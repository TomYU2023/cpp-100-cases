案例 81：std::nth_element 部分排序

使用 nth_element 在 O(N) 时间内找到第 K 大的元素，无需完整排序。

核心知识点：nth_element、部分排序、快速选择、中位数

完整代码（含逐行注释）

// 案例 81：std::nth_element 部分排序
// O(N) 时间找到第 K 大/小元素
#include <iostream>
#include <algorithm>
#include <vector>
#include <string>
using namespace std;

int main() {
    vector<int> data = {33, 10, 55, 22, 44, 11, 66, 88, 5, 77};

    cout << "原始数据: ";
    for (int x : data) cout << x << " ";
    cout << endl;

    // 1. 找到中位数（第 size/2 小的元素）
    auto v1 = data;
    auto mid = v1.begin() + v1.size() / 2;
    nth_element(v1.begin(), mid, v1.end());
    cout << "\n中位数: " << *mid << endl;
    cout << "nth_element 后: ";
    for (int x : v1) cout << x << " ";
    cout << endl;

    // 2. 找到第 3 小的元素
    auto v2 = data;
    auto kth = v2.begin() + 2; // 第 3 小（0-based）
    nth_element(v2.begin(), kth, v2.end());
    cout << "\n第 3 小: " << *kth << endl;

    // 3. 找到第 3 大的元素（用反向比较器）
    auto v3 = data;
    auto kthMax = v3.begin() + 2;
    nth_element(v3.begin(), kthMax, v3.end(), greater<int>());
    cout << "第 3 大: " << *kthMax << endl;

    // 4. 找到 Top 3（最小的 3 个元素）
    auto v4 = data;
    nth_element(v4.begin(), v4.begin() + 3, v4.end());
    cout << "\n最小的 3 个: ";
    for (int i = 0; i < 3; i++) cout << v4[i] << " ";
    cout << endl;

    // 5. 实际应用：找到分数排名的分界点
    vector<int> scores = {85, 92, 78, 95, 88, 72, 66, 90, 83, 76};
    int topN = 3;
    auto topBorder = scores.begin() + (scores.size() - topN);
    nth_element(scores.begin(), topBorder, scores.end());
    cout << "\nTop 3 分数线: " << *topBorder << " 分以上" << endl;

    return 0;
}

自定义比较器

【规则】sort/priority_queue/set 等 STL 容器和算法接受自定义比较器——函数指针、函数对象（functor）或 lambda 表达式。比较器必须满足严格弱序（strict weak ordering）。

【原理】比较器 comp(a, b) 返回 true 表示 a 应排在 b 前面。STL 的所有排序/查找算法都基于 < 操作构建，不依赖 ==。这是为了通用性——某些类型只有偏序关系（如任务优先级中可能有两个任务同等优先），严格全序反而难以建模。比较器需满足：irreflexive、transitive、比较等价传递性。

【常见陷阱】1. 将小于等于比较逻辑（<=）引入比较器，使得 comp(a,a) 返回 true——违反 irreflexive 原则，sort 会越界访问；2. 使用未初始化的变量作为排序依据——导致同一元素多次比较结果不同，sort 进入未定义行为。

nth_element(v.begin(), v.begin()+k, v.end(), greater<>());
// 第 k 大 = nth_element(..., greater<>())
// 等价于: 第 (n-k-1) 小
