案例 81：std::nth_element 部分排序

使用 nth_element 在 O(N) 时间内找到第 K 大的元素，无需完整排序。

核心知识点：nth_element、部分排序、快速选择、中位数

完整代码（含逐行注释）

// 案例 81：std::nth_element 部分排序
// O(N) 时间找到第 K 大/小元素
#include <iostream>
#include <algorithm>
#include <vector>
#include <string>
using namespace std;

int main() {
    vector<int> data = {33, 10, 55, 22, 44, 11, 66, 88, 5, 77};

    cout << "原始数据: ";
    for (int x : data) cout << x << " ";
    cout << endl;

    // 1. 找到中位数（第 size/2 小的元素）
    auto v1 = data;
    auto mid = v1.begin() + v1.size() / 2;
    nth_element(v1.begin(), mid, v1.end());
    cout << "\n中位数: " << *mid << endl;
    cout << "nth_element 后: ";
    for (int x : v1) cout << x << " ";
    cout << endl;

    // 2. 找到第 3 小的元素
    auto v2 = data;
    auto kth = v2.begin() + 2; // 第 3 小（0-based）
    nth_element(v2.begin(), kth, v2.end());
    cout << "\n第 3 小: " << *kth << endl;

    // 3. 找到第 3 大的元素（用反向比较器）
    auto v3 = data;
    auto kthMax = v3.begin() + 2;
    nth_element(v3.begin(), kthMax, v3.end(), greater<int>());
    cout << "第 3 大: " << *kthMax << endl;

    // 4. 找到 Top 3（最小的 3 个元素）
    auto v4 = data;
    nth_element(v4.begin(), v4.begin() + 3, v4.end());
    cout << "\n最小的 3 个: ";
    for (int i = 0; i < 3; i++) cout << v4[i] << " ";
    cout << endl;

    // 5. 实际应用：找到分数排名的分界点
    vector<int> scores = {85, 92, 78, 95, 88, 72, 66, 90, 83, 76};
    int topN = 3;
    auto topBorder = scores.begin() + (scores.size() - topN);
    nth_element(scores.begin(), topBorder, scores.end());
    cout << "\nTop 3 分数线: " << *topBorder << " 分以上" << endl;

    return 0;
}

语法规则：std::nth_element

nth_element(v.begin(), v.begin()+k, v.end());
// 将第 k 小的元素放在 v[k] 位置
// v[0..k-1] ≤ v[k] ≤ v[k+1..n-1]
// 平均 O(N)，最坏 O(N^2)（但实现通常用 Introselect 保证 O(N)）

举一反三：nth_element 基于快速选择算法（Quickselect）。比 sort + v[k] 快（O(N log N) vs O(N)）。常用于找中位数、topK、分数排名。不保证前后部分的顺序。

语法规则：部分排序与完整排序的选择

// 只需要第 K 个元素 → nth_element (O(N))
// 需要前 K 个元素但不关心顺序 → nth_element + 取前 K (O(N))
// 需要前 K 个元素且需顺序 → partial_sort (O(N log K))
// 需要全部排序 → sort (O(N log N))

举一反三：选择合适的算法避免过度排序。nth_element 是最优的「求第 K 个」方案。但注意 nth_element 后前 K 个元素是未排序的（仅保证都 ≤ v[k]）。

语法规则：自定义比较器

nth_element(v.begin(), v.begin()+k, v.end(), greater<>());
// 第 k 大 = nth_element(..., greater<>())
// 等价于: 第 (n-k-1) 小

举一反三：默认比较器 less<> 找第 K 小。greater<> 找第 K 大。配合自定义比较器可按任意条件界定「大小」。注意 0-based 索引和 1-based 第几的对应。
