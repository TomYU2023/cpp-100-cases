新语法讲解

运算符重载规则

C++ 允许重载大多运算符作为类成员或友元函数。不可重载的：`::` `.` `.*` `?:`。重载应保持运算符的语义直觉。
// 成员函数形式：返回类型 operator符号(参数)
// 全局函数形式：返回类型 operator符号(参数1, 参数2)

// 赋值运算符建议返回 *this 引用
Complex& operator=(const Complex& other) {
    real_ = other.real_; imag_ = other.imag_;
    return *this;
}

友元 friend

`friend` 声明让非成员函数或另一个类可以访问当前类的私有/保护成员。破坏封装但有时是必要的（如 `operator<<`）。
class A {
    int secret;
    friend void show(A& a);  // show 函数可以访问 A 的 private
};
void show(A& a) { cout << a.secret; }

this 指针

在成员函数内部，`this` 是指向当前对象的指针。`*this` 解引用得到对象本身。通常不需要显式使用，但返回自身引用时必须用 `return *this;`。
class Counter {
    int n;
public:
    Counter& increment() { n++; return *this; }
};
Counter c;
c.increment().increment();  // 链式调用
