案例 95：Raft 共识算法

实现 Raft 算法的核心选举模块：Follower/Candidate/Leader 状态转换，随机超时、投票、心跳机制。

核心知识点：Raft 选举状态机、std::mt19937 随机数、std::chrono 时间测量、分布式共识

完整代码（含逐行注释）

// 案例 95：Raft 共识算法（核心选举模块）
// Raft 是分布式系统核心算法（etcd/TiKV 使用）。
// 本案例实现 Leader 选举: Follower -> Candidate -> Leader。

#include <iostream>
#include <chrono>
#include <thread>
#include <random>
#include <vector>
#include <string>

enum class RaftState { FOLLOWER, CANDIDATE, LEADER };

class RaftNode {
    int id;
    RaftState state = RaftState::FOLLOWER;
    int currentTerm = 0;
    int votedFor = -1;
    int leaderId = -1;
    int voteCount = 0;
    int totalNodes;
    std::chrono::milliseconds electionTimeout;
    std::chrono::steady_clock::time_point lastHeartbeat;
    std::mt19937 rng;

public:
    RaftNode(int id, int total) : id(id), totalNodes(total), rng(std::random_device{}()) {
        resetElectionTimer();
        lastHeartbeat = std::chrono::steady_clock::now();
    }

    void resetElectionTimer() {
        std::uniform_int_distribution<int> dist(150, 300);
        electionTimeout = std::chrono::milliseconds(dist(rng));
    }

    bool isElectionTimeout() {
        auto now = std::chrono::steady_clock::now();
        auto elapsed = std::chrono::duration_cast<std::chrono::milliseconds>(now - lastHeartbeat);
        return elapsed >= electionTimeout;
    }

    void receiveHeartbeat(int term, int leader) {
        if (term >= currentTerm) {
            currentTerm = term;
            state = RaftState::FOLLOWER;
            leaderId = leader;
            votedFor = -1;
            lastHeartbeat = std::chrono::steady_clock::now();
            voteCount = 0;
            std::cout << "  [节点" << id << "] 收到心跳, Leader=" << leader << ", Term=" << term << std::endl;
        }
    }

    void startElection() {
        state = RaftState::CANDIDATE;
        currentTerm++;               // 任期 +1
        votedFor = id;               // 投票给自己
        voteCount = 1;               // 自己的一票
        resetElectionTimer();
        std::cout << "[节点" << id << "] 发起选举, Term=" << currentTerm << std::endl;
    }

    bool requestVote(int candidateId, int term) {
        if (term < currentTerm) return false;   // 候选人 term 更旧，拒绝
        if (term > currentTerm) {
            currentTerm = term;
            state = RaftState::FOLLOWER;
            votedFor = -1;
        }
        if (votedFor == -1 || votedFor == candidateId) {
            votedFor = candidateId;
            lastHeartbeat = std::chrono::steady_clock::now();
            std::cout << "  [节点" << id << "] 投票给节点" << candidateId << " (Term=" << term << ")" << std::endl;
            return true;
        }
        return false;
    }

    void receiveVote(bool granted, int term) {
        if (state != RaftState::CANDIDATE) return;
        if (term > currentTerm) {
            currentTerm = term;
            state = RaftState::FOLLOWER;
            voteCount = 0;
            return;
        }
        if (granted) {
            voteCount++;
            std::cout << "  [节点" << id << "] 获得一票 (" << voteCount << "/" << (totalNodes/2+1) << ")" << std::endl;
            if (voteCount > totalNodes / 2) becomeLeader();
        }
    }

    void becomeLeader() {
        state = RaftState::LEADER;
        leaderId = id;
        std::cout << "\n=== [节点" << id << "] 当选 Leader (Term=" << currentTerm << ") ===" << std::endl;
    }

    void sendHeartbeat(std::vector<RaftNode>& nodes) {
        if (state != RaftState::LEADER) return;
        for (auto& node : nodes)
            if (node.id != id)
                node.receiveHeartbeat(currentTerm, id);
    }

    int getId() const { return id; }
    RaftState getState() const { return state; }
    int getCurrentTerm() const { return currentTerm; }

    void tick(std::vector<RaftNode>& nodes) {
        if (state == RaftState::LEADER) {
            static int tickCount = 0;
            if (++tickCount % 5 == 0) sendHeartbeat(nodes);
        } else if (isElectionTimeout()) {
            startElection();
            for (auto& node : nodes)
                if (node.id != id) {
                    bool granted = node.requestVote(id, currentTerm);
                    receiveVote(granted, node.getCurrentTerm());
                }
        }
    }
};

int main() {
    const int N = 3;
    std::vector<RaftNode> nodes;
    for (int i = 0; i < N; i++)
        nodes.emplace_back(i, N);

    std::cout << "=== Raft 选举演示 (3 节点) ===" << std::endl;
    for (int t = 0; t < 50; t++) {
        for (auto& node : nodes) node.tick(nodes);
        std::this_thread::sleep_for(std::chrono::milliseconds(50));
        if (t % 10 == 0) {
            std::cout << "--- tick " << t << " ---" << std::endl;
            for (auto& node : nodes) {
                const char* states[] = {"Follower","Candidate","Leader"};
                std::cout << "  节点" << node.getId() << ": " << states[(int)node.getState()]
                          << " (Term=" << node.getCurrentTerm() << ")" << std::endl;
            }
        }
    }
    return 0;
}

语法规则：Raft 选举状态机模型

// Raft 的状态转换规则:
// Follower --(选举超时)--> Candidate --(获得多数票)--> Leader
// Candidate --(发现更高Term)--> Follower
// Leader --(发现更高Term)--> Follower

// 关键设计:
// 1. 随机选举超时(150~300ms) - 防止多个节点同时发起选举
// 2. 每个 Term 每节点只能投一票
// 3. 获得超过半数(N/2+1)投票即可当选 Leader

举一反三：Raft 是 Diego Ongaro 2014 年提出的共识算法，相比 Paxos 更易理解。选举只是 Raft 的一部分，完整实现还包括日志复制（Log Replication）和安全性约束。etcd、TiKV、Consul 等都使用 Raft。

语法规则：std::mt19937 随机数生成器

#include <random>
std::mt19937 rng(std::random_device{}());
std::uniform_int_distribution<int> dist(150, 300);
int timeout = dist(rng);  // 每次返回 [150,300] 随机整数

举一反三：C++11 的 库比 C 语言的 rand() 好得多：(1) 真正的均匀分布；(2) mt19937 周期长(2^19937-1)；(3) 支持多种分布(uniform/normal/bernoulli)。std::random_device 生成真随机种子。

语法规则：std::chrono 时间点差值

#include <chrono>
auto start = std::chrono::steady_clock::now();
auto now = std::chrono::steady_clock::now();
auto elapsed = std::chrono::duration_cast<std::chrono::milliseconds>(now - start);
if (elapsed >= std::chrono::milliseconds(300)) { /* 超时 */ }

举一反三：steady_clock 保证单调递增（不受系统时间调整影响），适合测量时间间隔。duration_cast 将高精度 duration 转为指定单位。
