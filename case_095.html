案例 95：std::bind 参数绑定

使用 bind 进行参数绑定和适配，理解占位符的使用。

核心知识点：bind、占位符、参数重排、ref/cref

完整代码（含逐行注释）

// 案例 95：std::bind 参数绑定
// 预绑定参数和重排参数顺序
#include <iostream>
#include <functional>
#include <vector>
#include <algorithm>
#include <string>
using namespace std;
using namespace placeholders;

// 一个接受多个参数的函数
void logMessage(const string& level, int code, const string& msg) {
    cout << "[" << level << "] " << code << ": " << msg << endl;
}

int add(int a, int b, int c) {
    return a + b + c;
}

bool isBetween(int value, int low, int high) {
    return value >= low && value <= high;
}

int main() {
    // 1. 绑定部分参数
    auto errorLog = bind(logMessage, "ERROR", _1, _2);
    auto warnLog  = bind(logMessage, "WARN",  _1, _2);

    errorLog(404, "页面未找到");
    warnLog(1001, "磁盘空间不足");

    // 2. 绑定全部参数
    auto hello = bind(logMessage, "INFO", 0, "系统启动完成");
    hello(); // 所有参数已绑定，无需传入

    // 3. 占位符重排参数顺序
    // add(a, b, c): 用 _2, _1, _3 交换前两个参数
    auto addSwapped = bind(add, _2, _1, _3);
    cout << "\naddSwapped(10, 20, 30) = "
         << addSwapped(10, 20, 30) << endl; // 20+10+30=60

    // 4. 固定某个参数的值
    auto addFive = bind(add, _1, 5, _2);
    cout << "addFive(10, 3) = " << addFive(10, 3) << endl; // 10+5+3=18

    // 5. 在算法中使用 bind
    vector<int> data = {1, 5, 10, 15, 20, 25, 30};

    // 统计在 [10, 20] 之间的元素
    auto inRange = bind(isBetween, _1, 10, 20);
    int count = count_if(data.begin(), data.end(), inRange);
    cout << "\n[10,20] 范围内的元素数: " << count << endl;

    // 6. ref / cref：绑定引用
    int counter = 0;
    auto increment = [](int& c) { c++; };

    // 对 bind，需要用 ref 传递引用
    auto incCounter = bind(increment, ref(counter));
    incCounter();
    incCounter();
    cout << "counter = " << counter << endl; // 2

    return 0;
}

语法规则：std::bind 基本用法

#include <functional>
using namespace placeholders;
auto f = bind(func, _1, _2, _3);
// _1, _2, ... 是调用 f(a,b,c) 时的参数位置
// 可绑定部分参数，固定其他参数的值

举一反三：bind 复制所有参数（值语义）。占位符 _1 对应新函数的第一个参数，_2 对应第二个。可用来创建参数更少的函数对象。C++11 后 Lambda 通常是更好的选择。

语法规则：占位符重排

auto f = bind(func, _2, _1, _3);
// f(a, b, c) → func(b, a, c)
// 可重复使用占位符： bind(func, _1, _1)  → func(a, a)
// 可跳过占位符：  bind(func, _1, 42, _2) → func(a, 42, b)

举一反三：占位符最多 _N（实现决定，通常 20+）。在 Lambda 中需手动写参数转发，而 bind 用占位符更简洁。N 个占位符对应 N 个新函数参数。

语法规则：ref / cref 引用绑定

int x = 0;
auto f = bind(func, ref(x)); // 传递引用
auto g = bind(func, cref(x)); // const 引用
// bind 默认复制参数，ref/cref 强制传递引用

举一反三：bind 内部存储的是参数的副本（值语义）。要修改外部变量必须用 ref。Lambda 通过捕获 [&] 或 [&x] 更自然地支持引用。这是 Lambda 优于 bind 的一个理由。
