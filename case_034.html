新语法讲解

chrono 三大时钟

system_clock：系统实时时钟，可转为日历时间，但可能被 NTP 或用户调整。steady_clock：单调递增时钟，不会被调整，适合测量时间间隔。high_resolution_clock：最高精度时钟（通常就是 steady_clock），写性能测试首选。

举一反三

// 测性能用 steady_clock/high_resolution_clock
auto t1 = steady_clock::now();
doWork();
auto t2 = steady_clock::now();
// 输出时间用 system_clock
auto now = system_clock::now();
time_t t = system_clock::to_time_t(now);

duration_cast 类型转换

时间差是一个 duration 模板类型。不同单位的 duration 之间用 duration_cast 转换（可能截断）或 floor/ceil/round（C++17）控制舍入方式。C++14 提供时间字面量：1s, 500ms, 100us, 50ns。

举一反三

auto d = 1500ms;
auto sec = duration_cast(d); // 1 (截断)
auto msec = duration_cast(d); // 1500
auto sec2 = floor(d); // C++17: 1
