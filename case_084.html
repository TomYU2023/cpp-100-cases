案例 84：std::unique/unique_copy 去重

使用 unique 去除连续重复元素，配合 sort 实现完全去重。

核心知识点：unique、unique_copy、erase-remove 惯用法、完全去重

完整代码（含逐行注释）

// 案例 84：std::unique / unique_copy 去重
// 去除相邻重复元素
#include <iostream>
#include <algorithm>
#include <vector>
#include <string>
#include <iterator>
using namespace std;

int main() {
    // 1. unique：去除连续重复
    vector<int> v1 = {1, 1, 2, 2, 2, 3, 1, 1, 4, 4, 5};
    cout << "原始: ";
    for (int x : v1) cout << x << " ";
    cout << endl;

    auto last = unique(v1.begin(), v1.end());
    v1.erase(last, v1.end()); // 删除尾部残留元素
    cout << "unique 后: ";
    for (int x : v1) cout << x << " ";
    cout << "\n(注意: 1 没有完全去除，因为不连续)" << endl;

    // 2. sort + unique = 完全去重
    vector<int> v2 = {3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5};
    cout << "\n原始: ";
    for (int x : v2) cout << x << " ";
    cout << endl;

    sort(v2.begin(), v2.end());
    auto last2 = unique(v2.begin(), v2.end());
    v2.erase(last2, v2.end());
    cout << "sort + unique 完全去重: ";
    for (int x : v2) cout << x << " ";
    cout << endl;

    // 3. unique_copy：去重并输出到新容器
    vector<int> v3 = {1, 1, 2, 2, 3, 3, 4, 4};
    vector<int> result;
    unique_copy(v3.begin(), v3.end(), back_inserter(result));
    cout << "\nunique_copy: ";
    for (int x : result) cout << x << " ";
    cout << endl;

    // 4. 自定义相等判断
    // 按绝对值去重
    vector<int> v4 = {-3, -1, 1, 2, -2, 3, 5};
    sort(v4.begin(), v4.end(),
         [](int a, int b) { return abs(a) < abs(b); });
    auto last4 = unique(v4.begin(), v4.end(),
        [](int a, int b) { return abs(a) == abs(b); });
    v4.erase(last4, v4.end());
    cout << "\n按绝对值去重: ";
    for (int x : v4) cout << x << " ";
    cout << endl;

    return 0;
}

语法规则：std::unique

auto last = unique(v.begin(), v.end());
v.erase(last, v.end());
// 将连续重复元素移到最后，返回新 end 迭代器
// 不真正删除元素（只是前移），需要 erase 配合

举一反三：这是 erase-remove 惯用法的一部分。unique 返回的是逻辑上的新结尾，实际元素还在后面。必须配合 erase 真正删除。只去连续重复。

语法规则：sort + unique 完全去重

sort(v.begin(), v.end());
auto last = unique(v.begin(), v.end());
v.erase(last, v.end());
// 排序把相同元素聚到一起，unique 去除连续重复

举一反三：两步完成完全去重。复杂度 O(N log N)（排序为主）。如果元素已有序，只需要 unique。unordered_set 可 O(N) 去重（但额外内存和不保证顺序）。

语法规则：unique_copy

unique_copy(src.begin(), src.end(), back_inserter(dst));
// 去重的同时复制到新容器
// 不修改原容器
// 结果也是相邻去重（通常配合 sorted 源）

举一反三：unique_copy 保持源容器不变。适合不可变数据或需要保留原始的情况。对有序源可完全去重。支持自定义比较器。
