案例 8：文本字符/单词频率统计

读取一段文本，统计每个字符和每个单词的出现频率，练习 map 和 stringstream 的使用。

核心知识点：map 统计、stringstream 分词、const 引用

完整代码（含逐行注释）

// 案例 8：文本字符 / 单词频率统计
// 使用 map 统计频率
#include <iostream>
#include <string>
#include <map>
#include <sstream>   // stringstream 用于分词
using namespace std;

int main() {
    string text = "hello world hello cpp world cpp cpp";

    // 1. 字符频率统计
    map<char, int> charCount;
    for (char c : text) {
        if (c != ' ') {           // 忽略空格
            charCount[c]++;       // 自动初始化为 0 并 +1
        }
    }

    cout << "=== 字符频率 ===" << endl;
    for (const auto& pair : charCount) {
        cout << "'" << pair.first << "': " << pair.second << endl;
    }

    // 2. 单词频率统计
    map<string, int> wordCount;
    stringstream ss(text);
    string word;
    while (ss >> word) {          // 按空白字符分词
        wordCount[word]++;
    }

    cout << "\n=== 单词频率 ===" << endl;
    for (const auto& pair : wordCount) {
        cout << pair.first << ": " << pair.second << endl;
    }

    return 0;
}

语法规则：map 统计频率

#include <map>
map<string, int> freq;
freq[word]++; // 键不存在时自动插入并初始化为 0

举一反三：map 基于红黑树，有序；如需更高性能可用 unordered_map（哈希表）。freq[key] 不存在时自动创建，用 freq.at(key) 则不存在时抛异常。

语法规则：stringstream 分词

#include <sstream>
stringstream ss(text);
string word;
while (ss >> word) { /* 处理 */ }

举一反三：stringstream 默认按空白分割，也可用 getline(ss, token, ',') 按指定分隔符分割。还可用于字符串和其他类型的转换（ss >> num）。

语法规则：基于范围的 for + const auto&

for (const auto& pair : wordCount) {
    cout << pair.first << ": " << pair.second << endl;
}

举一反三：const auto& 避免拷贝 pair，提升性能。map 中 pair.first 是键、pair.second 是值。C++17 可用结构化绑定：for (const auto& [k,v] : m)。
