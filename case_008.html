新语法讲解

文件流 fstream

三种文件流：`ifstream`（读）、`ofstream`（写）、`fstream`（读写）。文件打开后可像 cin/cout 一样操作。
ofstream out("output.txt");
out << "Hello File!" << endl;
out.close();

std::map

`map` 是基于红黑树的有序关联容器。键唯一，按键的 `<` 运算符排序。插入/查找/删除都是 O(log n)。`[]` 访问时键不存在会自动插入默认值。
map scores;
scores["Alice"] = 95;
cout << scores["Bob"];  // 输出 0，会自动插入 Bob

auto 类型推导

`auto` 让编译器自动推导变量类型。C++11 引入，能减少冗长类型名。结合 `const&` 可避免拷贝。
auto x = 42;         // int
auto y = 3.14;       // double
auto s = "hello"s;   // std::string（需要 using namespace std::literals）
