案例 100：终点即起点

语法规则：回顾 std::initializer_list 聚合初始化

std::vector<SkillCategory> skills = {
    {"语言核心", {"指针", "模板", "Lambda"}},
    {"并发", {"mutex", "thread"}}
};
// vector 接受 initializer_list<SkillCategory>
// SkillCategory 接受 initializer_list<string>（聚合初始化）

举一反三：C++11 的 initializer_list 让容器初始化变得优雅。本质是编译器在栈上生成临时数组，然后传给构造函数。注意 initializer_list 只持有常量引用，不拥有数据，出了作用域就失效。
