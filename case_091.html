案例 91：模板特化与偏特化

理解函数模板完全特化、类模板全特化与偏特化的规则和应用场景。

核心知识点：全特化、偏特化、优先级、SFINAE 入门

完整代码（含逐行注释）

// 案例 91：模板特化与偏特化
// 为特定类型或类型族提供定制实现
#include <iostream>
#include <string>
#include <type_traits>
using namespace std;

// 1. 主模板
template <typename T>
struct TypeInfo {
    static const char* name() { return "未知类型"; }
};

// 2. 全特化（针对特定类型）
template <>
struct TypeInfo<int> {
    static const char* name() { return "整数 (int)"; }
};

template <>
struct TypeInfo<double> {
    static const char* name() { return "双精度浮点 (double)"; }
};

template <>
struct TypeInfo<string> {
    static const char* name() { return "字符串 (string)"; }
};

// 3. 偏特化：针对所有指针类型
template <typename T>
struct TypeInfo<T*> {
    static string name() {
        return string("指向 ") + TypeInfo<T>::name() + " 的指针";
    }
};

// 4. 偏特化：针对所有 const 类型
template <typename T>
struct TypeInfo<const T> {
    static string name() {
        return string("const ") + TypeInfo<T>::name();
    }
};

int main() {
    cout << "int: " << TypeInfo<int>::name() << endl;
    cout << "double: " << TypeInfo<double>::name() << endl;
    cout << "char: " << TypeInfo<char>::name() << endl;
    cout << "string: " << TypeInfo<string>::name() << endl;
    cout << "int*: " << TypeInfo<int*>::name() << endl;
    cout << "const double: " << TypeInfo<const double>::name() << endl;
    cout << "const int*: " << TypeInfo<const int*>::name() << endl;
    return 0;
}

语法规则：全特化 vs 偏特化

// 全特化: template<> 为特定类型提供定制
template <> struct MyClass<int> { ... };
// 偏特化: 为类型族提供定制（指针/const/引用）
template <typename T> struct MyClass<T*> { ... };

举一反三：全特化实例化时完全匹配。偏特化覆盖一类类型。匹配规则：全特化优先级最高，然后最具体的偏特化，最后主模板。函数模板只支持全特化。

语法规则：偏特化模式

// T* : 所有指针类型
template <typename T> struct S<T*> { ... };
// const T : 所有 const 类型
template <typename T> struct S<const T> { ... };
// T& : 所有引用类型

举一反三：偏特化不能用于函数模板（但可用重载代替）。偏特化是模板元编程的基础（配合 SFINAE）。常见模式：移除 const/引用（decay）、提取元素类型。

语法规则：匹配规则与歧义

// 当多个偏特化都匹配时，选择「更特化」的那个
// 如 const int* 匹配 T* 也匹配 const T
// 但 const T* 比两者都更特化
// 编译器选择最特化的匹配

举一反三：匹配遵循「最特化优先」原则。如果无法确定哪个最特化，编译报错（歧义）。了解匹配规则有助于设计正确的特化层次。
