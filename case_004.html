新语法讲解

布尔类型 bool

`bool` 只有两个值：`true` 和 `false`。函数返回值类型为 bool 表示这个函数回答「是或否」。
bool isEven(int n) { return n % 2 == 0; }

短路求值

C++ 的逻辑运算符 `&&` 和 `||` 使用短路求值：`A && B` 中若 A 为 false 则不再计算 B；`A || B` 中若 A 为 true 则不再计算 B。
if (ptr != nullptr && ptr->value > 0)  // 安全：ptr 空时不会解引用

sqrt() 与 cmath

`sqrt(x)` 返回 x 的平方根。需要 `#include `。素数判断中 `i<=sqrt(n)` 等价于 `i*i<=n`，后者避免了浮点精度问题。
double root = sqrt(16.0);  // root = 4.0
