新语法讲解

函数模板 template

语法：template <typename T> 后跟函数定义。T 是占位符，编译时根据调用参数的实际类型自动生成对应版本（叫模板实例化）。
template<typename T>
T add(T a, T b) { return a + b; }
add(1, 2);    // 生成 int 版本
add(1.0, 2.0); // 生成 double 版本

值初始化 T()

T() 产生 T 类型的零值：数值类型为 0/0.0，指针为 nullptr，类类型为默认构造对象。用于泛型编程中的初始化。
int i = int();     // 0
double d = double(); // 0.0
string s = string(); // 空串

举一反三

模板特化：字符串拼接 sum

代码：template<> string sum(string arr[], int n) { ... }

思路：对特定类型提供不同实现

支持多个类型参数

代码：template auto multiply(T a, U b) -> decltype(a*b);

思路：decltype 推导返回类型

非类型模板参数

代码：template T sum(T (&arr)[N]) { ... }

思路：用模板参数自动推导数组大小
