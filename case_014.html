案例 14：模板函数

编写通用模板函数，实现任意类型的最大值查找、数组打印、交换等功能，理解泛型编程基础。

核心知识点：函数模板、类型推导、模板特化

完整代码（含逐行注释）

// 案例 14：模板函数
// 练习函数模板的定义和使用
#include <iostream>
#include <string>
using namespace std;

// 模板函数：返回两个值中较大的那个
template <typename T>
T myMax(T a, T b) {
    return (a > b) ? a : b;
}

// 模板函数：打印数组
template <typename T>
void printArray(const T arr[], size_t size) {
    cout << "[ ";
    for (size_t i = 0; i < size; i++) {
        cout << arr[i];
        if (i < size - 1) cout << ", ";
    }
    cout << " ]" << endl;
}

// 模板函数：交换两个值（引用传递）
template <typename T>
void mySwap(T& a, T& b) {
    T temp = a;
    a = b;
    b = temp;
}

int main() {
    // 类型自动推导
    cout << "max(10, 20) = " << myMax(10, 20) << endl;
    cout << "max(3.14, 2.72) = " << myMax(3.14, 2.72) << endl;
    cout << "max(\"abc\", \"abd\") = " << myMax(string("abc"), string("abd")) << endl;

    // 打印数组
    int intArr[] = {1, 2, 3, 4, 5};
    double dblArr[] = {1.1, 2.2, 3.3};
    printArray(intArr, 5);
    printArray(dblArr, 3);

    // 交换
    int x = 10, y = 20;
    mySwap(x, y);
    cout << "交换后: x=" << x << ", y=" << y << endl;

    return 0;
}

语法规则：函数模板定义

template <typename T>
T myMax(T a, T b) {
    return (a > b) ? a : b;
}

举一反三：typename 和 class 关键字在模板参数中等价。编译器根据调用实参自动推导 T 的类型，也可显式指定：myMax(3, 5)。

语法规则：类型推导与显式指定

myMax(3, 5);      // 推导为 int
myMax(3.0, 5.0);  // 推导为 double
myMax<int>(3, 5); // 显式指定

举一反三：类型推导要求所有参数类型一致，否则需显式指定或使用多个模板参数。模板参数不限于类型，也可是非类型参数如 template 。

语法规则：引用传递模板

template <typename T>
void mySwap(T& a, T& b) { /* ... */ }

举一反三：使用引用参数可修改实参。对于大对象用 const T& 避免拷贝。泛型编程中引用传递是常见模式。
