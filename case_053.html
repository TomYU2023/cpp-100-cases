    新语法精讲

    C++11 原始字符串字面量 R"(...)"

    语法规则：Raw String Literal R"( ... )" 内的所有字符保持原样，反斜杠不会转义，换行直接保留。非常适合嵌入 HTML、JSON、正则表达式等多行文本。分隔符可自定义：R"delim( ... )delim"。
    // 原始字符串：不需要转义反斜杠和引号
string json = R"(
{
  "name": "Alice",
  "age": 25
}
)";
// 等效的普通字符串（需要大量转义）
string json2 = "\n{\n  \"name\": \"Alice\",\n  \"age\": 25\n}\n";

    std::regex 正则表达式基础

    语法规则：regex_match（完全匹配）、regex_search（部分匹配）、regex_replace（替换）。构造 regex 对象较慢，应尽量复用。原始字符串特别适合写正则表达式。
    regex boldPattern(R"(\*\*(.+?)\*\*)");
string text = "Hello **World**!";
string result = regex_replace(text, boldPattern, "$1");
// result = "Hello World!"

    istringstream：字符串当作流来读

    语法规则：istringstream 将 string 包装为输入流，可以像读文件一样用 >> 或 getline 读取。适合需要按行/按词解析字符串的场景。
    string data = "apple\nbanana\ncherry";
istringstream ss(data);
string line;
while (getline(ss, line)) {
    cout << line << endl;
