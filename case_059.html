新语法精讲

PROCESS_MEMORY_COUNTERS 结构体

语法规则：Windows 进程内存的核心统计：WorkingSetSize（当前物理内存占用）、PeakWorkingSetSize（历史峰值）、PagefileUsage（页面文件使用）、PageFaultCount（缺页次数）。通过 GetProcessMemoryInfo 获取。
PROCESS_MEMORY_COUNTERS pmc{};
GetProcessMemoryInfo(GetCurrentProcess(), &pmc, sizeof(pmc));
// 当前进程的内存：
cout << "Working Set: " << pmc.WorkingSetSize / 1024 << " KB";

EnumProcesses 进程枚举

语法规则：EnumProcesses 获取系统所有进程 PID 到数组。返回数组大小需用 bytesReturned / sizeof(DWORD) 计算进程数。注意 PID 可重用，枚举到的进程可能在调用前已退出。
DWORD pids[4096], cb;
EnumProcesses(pids, sizeof(pids), &cb);
int count = cb / sizeof(DWORD);
for (int i = 0; i < count; i++) {
    HANDLE h = OpenProcess(PROCESS_QUERY_INFORMATION, FALSE, pids[i]);
    // ... 查询进程信息 ...
    CloseHandle(h);
}

举一反三

  • 跨平台进程查询（Linux /proc）：
    // Linux: 读取 /proc/[pid]/statm
ifstream f("/proc/self/status");
string line;
while (getline(f, line)) {
    if (line.starts_with("VmRSS:")) ss >> mem;
}

    Linux 通过 /proc 文件系统提供进程信息，VmRSS 对应 Windows 的 WorkingSet。跨平台工具通常用条件编译区分。

  • 实时监控 + 告警：
    while (true) {
    auto& hwInfo = getHardwareInfo();
    double memUsage = hwInfo.usedMemory / hwInfo.totalMemory;
    if (memUsage > 0.95) alert("Memory usage > 95%!");
    this_thread::sleep_for(5s);
}

    可以扩展为一个系统资源监控面板，结合条件变量或回调在超过阈值时触发告警。
